인스타그램 릴스나 게시글 등에서 마음에 드는 장소를 발견했을 때 저장하고는 정리를 하지 않아서 관리가 어렵다.
또한 저장한 릴스/게시글의 정보를 다시 확인해서 지도맵으로 직접 확인해야 하는 것도 귀찮다.
그리고 릴스/게시글이 광고글인지도 확인이 어렵기 때문에 검증이 필요하다.

릴스/게시글 저장 혹은 공유 → 데이터 파싱 → 장소명, 주소, 특징 등을 분류 → 지도 3 종류 (구글맵, 네이버맵, 카카오맵)에서 검색 → 장소명, 유형, 주소, 영업시간, 휴무일, 특징을 파싱 → 리뷰 내용 분석 → 검증도 0~1 점수 산출 → json 형태로 저장
사용자가 오늘 뭐할지 찾는다 → 유형별 검색 → 지도에 먼저 띄워줌 → 랜덤 or 사용자 직접 선택
