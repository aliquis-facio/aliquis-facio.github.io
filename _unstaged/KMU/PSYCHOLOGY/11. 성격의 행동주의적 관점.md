## 1. 11주차 전체 주제

11주차의 핵심 주제는 **성격을 행동주의 관점에서 어떻게 이해할 수 있는가**이다. 행동주의는 인간의 성격을 타고난 내면 구조나 무의식의 산물로 보기보다, **환경 속에서 학습된 행동 패턴의 집합**으로 본다. 즉, 한 사람이 반복적으로 보이는 성격적 특징은 과거에 어떤 자극을 경험했고, 어떤 행동이 보상 또는 처벌을 받았는가에 따라 형성된 결과이다.

# 2. 행동주의 관점의 기본 전제

## 2.1 성격은 학습된 행동 패턴이다

행동주의에서 성격은 “내 안에 있는 고정된 본질”이 아니라, **경험을 통해 학습된 행동 경향성**이다. 어떤 사람이 외향적이거나 내향적이고, 논쟁적이거나 조용하며, 불안을 잘 느끼거나 회피적인 태도를 보이는 것은 그 행동들이 과거 환경 속에서 강화되거나 처벌되었기 때문이라고 본다. 

강의에서는 이를 다음과 같이 설명했다.

> 성격은 습관처럼 형성된 학습된 패턴이다.
> 인간은 경험을 통해 행동을 획득하고 변화한다.
> 따라서 성격도 학습의 결과이다. 

심리학에서 말하는 **학습**은 단순히 공부하는 것이 아니다. 경험을 통해 행동, 감정, 반응 방식이 변화하는 모든 과정을 의미한다.

---

## 2.2 행동주의는 관찰 가능한 행동에 집중한다

행동주의는 심리학을 과학적으로 만들고자 한 관점이다. 그래서 눈에 보이지 않는 무의식, 내면 구조, 자기실현 욕구보다는 **관찰 가능하고 측정 가능한 행동**을 연구 대상으로 삼는다. 

행동주의의 기본 공식은 다음과 같다.

| 요소 | 의미           |
| -- | ------------ |
| S  | Stimulus, 자극 |
| R  | Response, 반응 |

즉, 행동주의는 **자극이 주어졌을 때 어떤 반응이 나타나는가**, 그리고 **그 반응이 어떤 결과에 의해 유지되거나 사라지는가**를 분석한다.

행동주의에서는 인간의 내면을 일종의 블랙박스로 두고, 외부에서 관찰 가능한 행동을 중심으로 설명한다. 그래서 “왜 그런 마음을 가졌는가?”보다 “그 행동이 어떤 자극과 결과 속에서 학습되었는가?”를 묻는다.

# 3. 행동주의의 세 가지 학습 방법

행동주의에서 성격과 행동을 설명하는 대표적 학습 방법은 세 가지이다.

| 학습 방법    | 대표 학자    | 핵심 원리       | 성격 형성과의 관련              |
| -------- | -------- | ----------- | ----------------------- |
| 고전적 조건형성 | 파블로프, 왓슨 | 자극과 자극의 연합  | 공포, 불안, 호감 같은 정서 반응 형성  |
| 조작적 조건형성 | 스키너      | 행동과 결과의 연합  | 반복 행동, 습관, 성격적 행동 패턴 형성 |
| 사회적 학습   | 반두라      | 관찰, 모방, 모델링 | 타인의 행동을 보고 배우는 성격 형성    |

11주차에서는 특히 **고전적 조건형성**과 **조작적 조건형성**이 중심이었다.

# 4. 고전적 조건형성

## 4.1 개념

고전적 조건형성은 **자극과 자극이 반복적으로 연결되어 새로운 반응이 나타나는 학습 과정**이다. 대표적인 예는 파블로프의 개 실험이다. 개는 원래 음식을 보면 침을 흘린다. 그런데 음식과 종소리를 반복적으로 함께 제시하면, 나중에는 음식이 없어도 종소리만 듣고 침을 흘리게 된다. 

| 단계     | 자극       | 반응    |
| ------ | -------- | ----- |
| 조건형성 전 | 음식       | 침 분비  |
| 조건형성 전 | 종소리      | 반응 없음 |
| 조건형성 중 | 종소리 + 음식 | 침 분비  |
| 조건형성 후 | 종소리      | 침 분비  |

이때 종소리는 처음에는 아무 반응도 일으키지 않는 **중성자극**이었지만, 음식과 반복적으로 연결된 후에는 침 분비를 일으키는 **조건자극**이 된다.

---

## 4.2 주요 용어

| 용어        | 의미                      | 예시           |
| --------- | ----------------------- | ------------ |
| 무조건자극, US | 원래부터 반응을 일으키는 자극        | 음식           |
| 무조건반응, UR | 무조건자극에 의해 자연스럽게 나타나는 반응 | 침 분비         |
| 중성자극, NS  | 처음에는 특별한 반응을 일으키지 않는 자극 | 종소리          |
| 조건자극, CS  | 학습 후 반응을 일으키는 자극        | 종소리          |
| 조건반응, CR  | 조건자극에 의해 나타나는 학습된 반응    | 종소리에 대한 침 분비 |

강의에서는 중성자극이 무조건자극과 반복적으로 연합되면서 “힘을 얻는다”고 설명했다. 즉, 원래 아무 의미 없던 자극이 특정 감정이나 반응을 유발하는 자극으로 바뀌는 것이다. 

---

## 4.3 고전적 조건형성과 정서 반응

고전적 조건형성은 특히 **공포, 불안, 호감, 혐오** 같은 정서 반응을 설명하는 데 유용하다.

예를 들어 치과라는 건물 자체는 원래 무서운 자극이 아니다. 그러나 치과에서 아픈 치료를 받은 경험이 반복되면, 치과 건물이나 치과 냄새만으로도 불안이 생길 수 있다. 이 경우 치과는 조건자극이 되고, 불안은 조건반응이 된다.

| 사례       | 무조건자극    | 무조건반응     | 조건자극        | 조건반응   |
| -------- | -------- | --------- | ----------- | ------ |
| 치과 공포    | 아픈 치료    | 공포, 불안    | 치과          | 불안     |
| 악몽을 꾼 침대 | 악몽       | 두려움       | 침대          | 무서움    |
| 넘어진 장소   | 신체적 충격   | 놀람, 창피함   | 그 장소        | 긴장, 불안 |
| 운동 음악    | 운동 후 개운함 | 활력, 기분 좋음 | 운동할 때 들은 음악 | 신남     |

이처럼 고전적 조건형성은 “왜 특정 장소, 사람, 소리, 냄새가 특정 감정을 불러일으키는가”를 설명한다.

# 5. 공포조건화와 자극 일반화

## 5.1 공포조건화

공포조건화는 원래 중립적이던 자극이 불쾌하거나 위협적인 자극과 연결되어 공포 반응을 일으키게 되는 과정이다.

대표적인 실험은 왓슨의 **리틀 앨버트 실험**이다. 처음에 어린 아이 앨버트는 흰쥐를 무서워하지 않았다. 하지만 흰쥐를 보여줄 때마다 큰 소리를 함께 들려주자, 나중에는 흰쥐만 보고도 공포 반응을 보이게 되었다. 

| 요소    | 리틀 앨버트 실험 |
| ----- | --------- |
| 중성자극  | 흰쥐        |
| 무조건자극 | 큰 소리      |
| 무조건반응 | 놀람, 공포    |
| 조건자극  | 흰쥐        |
| 조건반응  | 흰쥐에 대한 공포 |

이 실험은 인간의 공포가 반드시 타고나는 것이 아니라 **학습될 수 있다**는 점을 보여준다.

---

## 5.2 자극 일반화

자극 일반화는 조건반응이 원래의 조건자극과 비슷한 다른 자극으로 확장되는 현상이다. 리틀 앨버트는 흰쥐뿐 아니라 흰토끼, 털이 있는 대상에도 공포 반응을 보였다. 

강의에서는 이를 “자라 보고 놀란 가슴 솥뚜껑 보고 놀란다”라는 속담으로 설명했다. 

예를 들어 새와 관련된 충격적인 경험을 한 사람이 이후 새뿐 아니라 깃털, 날갯짓, 비둘기, 깃털 장식까지 불편하게 느낀다면 이는 자극 일반화로 볼 수 있다.

---

# 6. 역조건화

역조건화는 이미 형성된 공포나 부정적 반응을 새로운 긍정적 반응으로 바꾸는 과정이다. 공포 대상과 긍정적인 자극을 함께 제시하여, 기존 공포 반응을 약화시키는 방식이다. 

대표 사례는 피터라는 아이의 토끼 공포 감소 사례이다. 피터는 토끼를 무서워했지만, 좋아하는 음식을 먹는 상황에서 토끼를 점진적으로 가까이 두자 토끼에 대한 공포가 줄어들었다.

| 기존 반응    | 역조건화 방식      | 결과                |
| -------- | ------------ | ----------------- |
| 토끼 → 공포  | 토끼 + 좋아하는 음식 | 토끼에 대한 공포 감소      |
| 청소기 → 공포 | 청소기 + 간식     | 청소기에 대한 긍정적 반응 형성 |
| 병원 → 불안  | 병원 + 안전한 경험  | 병원 불안 감소          |

역조건화의 핵심은 공포보다 더 강한 긍정적 반응을 유발하는 자극을 함께 제시하는 것이다.

# 7. 조작적 조건형성

## 7.1 개념

조작적 조건형성은 **행동과 그 행동의 결과가 연결되어 행동이 증가하거나 감소하는 과정**이다. 고전적 조건형성이 자극과 자극의 연합이라면, 조작적 조건형성은 행동과 결과의 연합이다. 

기본 구조는 다음과 같다.

> 행동 → 결과 → 행동 증가 또는 감소

즉, 어떤 행동을 했을 때 좋은 결과가 오면 그 행동은 증가하고, 나쁜 결과가 오면 그 행동은 감소한다.

---

## 7.2 스키너 상자

조작적 조건형성의 대표 학자는 스키너이다. 스키너는 쥐가 레버를 누르면 먹이가 나오는 장치를 통해 행동과 결과의 관계를 연구했다. 처음에 쥐는 우연히 레버를 누르지만, 레버를 누르면 먹이가 나온다는 것을 학습하면 이후에는 의도적으로 레버를 누르게 된다. 

| 행동     | 결과    | 이후 변화     |
| ------ | ----- | --------- |
| 레버 누르기 | 먹이 제공 | 레버 누르기 증가 |

이 원리는 인간 행동에도 적용된다. 질문했을 때 좋은 답변을 얻으면 질문 행동이 증가하고, 무시당하거나 창피를 당하면 질문 행동이 줄어든다.

---

# 8. 행동조성

행동조성은 원하는 목표 행동이 처음부터 나타나기 어려울 때, 목표 행동에 가까운 행동을 단계적으로 강화하여 최종 행동을 만들어 가는 과정이다. 

예를 들어 아이에게 하루 1시간 공부하는 습관을 만들고 싶다면 처음부터 1시간을 요구하지 않는다.

| 단계  | 목표 행동    | 강화       |
| --- | -------- | -------- |
| 1단계 | 5분 앉아 있기 | 칭찬, 보상   |
| 2단계 | 10분 공부하기 | 칭찬, 보상   |
| 3단계 | 20분 공부하기 | 칭찬, 보상   |
| 최종  | 1시간 공부하기 | 목표 달성 보상 |

강의에서는 마켓컬리 신규 가입 혜택, 첫 구매 쿠폰, 무료배송, 등급 상승 쿠폰도 행동조성의 예로 설명되었다. 소비자가 처음에는 작은 보상으로 구매를 시작하고, 이후 더 많은 구매 행동으로 유도되는 구조이다.

# 9. 강화와 처벌

## 9.1 기본 구분

조작적 조건형성에서 가장 중요한 개념은 강화와 처벌이다.

| 개념 | 의미           |
| -- | ------------ |
| 강화 | 행동을 증가시키는 결과 |
| 처벌 | 행동을 감소시키는 결과 |
| 정적 | 자극을 제공함      |
| 부적 | 자극을 제거함      |

여기서 주의할 점은 **정적은 좋은 것**, **부적은 나쁜 것**이라는 뜻이 아니라는 점이다. 정적은 자극을 더하는 것이고, 부적은 자극을 제거하는 것이다. 

---

## 9.2 네 가지 유형

| 구분    | 의미                | 예시               |
| ----- | ----------------- | ---------------- |
| 정적 강화 | 좋은 자극을 제공하여 행동 증가 | 방 청소를 하면 용돈을 줌   |
| 부적 강화 | 싫은 자극을 제거하여 행동 증가 | 안전벨트를 매면 경고음이 멈춤 |
| 정적 처벌 | 싫은 자극을 제공하여 행동 감소 | 주차 위반 시 벌금 부과    |
| 부적 처벌 | 좋은 자극을 제거하여 행동 감소 | 잘못된 행동 후 게임 금지   |

간단히 정리하면 다음과 같다.

| 결과    | 자극 제공 | 자극 제거 |
| ----- | ----- | ----- |
| 행동 증가 | 정적 강화 | 부적 강화 |
| 행동 감소 | 정적 처벌 | 부적 처벌 |

---

# 10. 강화계획

## 10.1 강화계획의 의미

강화계획은 보상을 **언제, 어떤 방식으로 제공할 것인가**에 대한 계획이다. 같은 보상이라도 매번 주는지, 가끔 주는지, 정해진 횟수 후에 주는지, 예측 불가능하게 주는지에 따라 행동의 지속성과 소거 저항이 달라진다. 

강화계획은 크게 두 가지이다.

| 구분     | 의미            |
| ------ | ------------- |
| 연속 강화  | 행동할 때마다 매번 보상 |
| 간헐적 강화 | 가끔 보상         |

---

## 10.2 연속 강화

연속 강화는 목표 행동이 나타날 때마다 보상을 주는 방식이다. 새로운 행동을 처음 학습시키는 데 효과적이다. 하지만 보상이 사라지면 행동도 빠르게 소거될 수 있다.

예를 들어 버튼을 누를 때마다 먹이가 나오던 동물이 어느 순간 먹이가 나오지 않으면, 버튼 누르기 행동을 빠르게 멈출 수 있다.

---

## 10.3 간헐적 강화

간헐적 강화는 매번 보상을 주지 않고 일정한 기준에 따라 가끔 보상하는 방식이다. 행동을 오래 유지시키는 데 효과적이다.

| 유형      | 기준 | 설명                | 예시                    |
| ------- | -- | ----------------- | --------------------- |
| 고정비율 강화 | 횟수 | 정해진 횟수 후 보상       | 커피 쿠폰 10개, 칭찬 스티커     |
| 변동비율 강화 | 횟수 | 몇 번 해야 보상이 나올지 모름 | 슬롯머신, 랜덤 뽑기, SNS 새로고침 |
| 고정간격 강화 | 시간 | 정해진 시간 후 보상       | 월급, 정기시험              |
| 변동간격 강화 | 시간 | 언제 보상이 올지 모름      | 깜짝 퀴즈, 갑작스러운 세일       |

---

## 10.4 변동비율 강화의 강력함

수업에서 특히 강조된 것은 **변동비율 강화**이다. 변동비율 강화는 보상이 언젠가는 나오지만, 몇 번 행동해야 나오는지 알 수 없는 방식이다. 이 방식은 행동을 가장 강하게 유지시킨다.

예시는 다음과 같다.

| 사례      | 변동비율 구조                 |
| ------- | ----------------------- |
| 슬롯머신    | 몇 번 당겨야 당첨될지 모름         |
| 포켓몬 띠부씰 | 몇 개를 사야 원하는 스티커가 나올지 모름 |
| 랜덤 장난감  | 몇 번 사야 원하는 장난감이 나올지 모름  |
| SNS 쇼츠  | 몇 번 넘겨야 재미있는 영상이 나올지 모름 |
| 스마트폰 알림 | 언제 원하는 메시지가 올지 모름       |

스마트폰과 SNS는 이 구조를 매우 잘 활용한다. 사용자는 매번 보상을 받지는 않지만, 가끔 흥미로운 영상, 메시지, 댓글, 좋아요를 얻는다. 그래서 “한 번만 더” 확인하게 된다.

# 11. 스마트폰 중독의 행동주의적 분석

11-1 논문은 스마트폰 중독을 행동주의의 **강화계획** 원리로 분석한다. 스마트폰은 사용자가 메시지, SNS 반응, 재미있는 콘텐츠, 유용한 정보 같은 보상을 예측 불가능하게 얻게 만드는 매체이다. 따라서 스마트폰 사용은 정적 강화에 의해 유지되며, 특히 변동비율 강화 또는 변동간격 강화의 구조를 갖는다. 

## 11.1 스마트폰 사용의 보상 구조

| 스마트폰 행동 | 가능한 보상         |
| ------- | -------------- |
| 카카오톡 확인 | 친구의 메시지        |
| SNS 확인  | 좋아요, 댓글, 새 게시물 |
| 쇼츠 넘기기  | 흥미로운 영상        |
| 이메일 확인  | 중요한 메일         |
| 알림 확인   | 유용한 정보         |

이 보상은 매번 제공되지 않는다. 어떤 때는 원하는 메시지가 오고, 어떤 때는 아무것도 없다. 그러나 가끔 보상이 주어지기 때문에 사용자는 계속 확인한다. 이것이 스마트폰 중독의 행동주의적 핵심이다.

---

## 11.2 스마트폰 중독의 특징

논문은 스마트폰 중독을 행동중독의 하나로 본다. 행동중독은 도박, 강박적 구매, 인터넷 중독처럼 특정 행동을 반복적으로 통제하지 못하는 상태이다. 스마트폰 중독은 약물중독처럼 갈망, 내성, 금단, 일상생활 장애를 보일 수 있다. 

| 증상      | 의미                  |
| ------- | ------------------- |
| 갈망      | 스마트폰을 계속 확인하고 싶은 욕구 |
| 내성      | 점점 더 오래 사용해야 만족     |
| 금단      | 사용하지 못하면 불안, 초조     |
| 일상생활 장애 | 학업, 수면, 대인관계 문제     |

---

## 11.3 해결 방안: 고정 강화계획으로 전환

논문은 스마트폰 중독을 줄이기 위해 **가변 강화계획을 고정 강화계획으로 바꾸는 전략**을 제안한다. 즉, 스마트폰을 아무 때나 확인하는 것이 아니라 정해진 시간과 횟수에만 사용하도록 구조화하는 것이다. 

| 방식    | 예시               |
| ----- | ---------------- |
| 고정간격  | 4시간마다 한 번 확인     |
| 고정비율  | 하루 4회만 확인        |
| 시간 제한 | 한 번 사용할 때 30분 이하 |

이 전략은 스마트폰 사용을 예측 가능한 구조로 바꾸어 강박적 확인 행동을 줄이는 데 목적이 있다.

# 12. 스마트폰 중독의 행동신경과학적 분석

11-1 논문은 행동주의만으로는 스마트폰 중독의 내적 기제를 충분히 설명하기 어렵다고 보고, 행동신경과학적 관점도 함께 제시한다. 핵심은 **보상회로**와 **전전두피질(PFC)**이다. 

## 12.1 주요 뇌 영역

| 뇌 영역       | 기능                  |
| ---------- | ------------------- |
| 전전두피질, PFC | 판단, 계획, 자기조절, 충동 억제 |
| 복측피개야, VTA | 도파민 보상회로 출발         |
| 측핵         | 보상, 쾌감, 강화학습        |
| 편도체        | 공포, 불안, 공격성         |
| 변연계        | 정서와 동기 조절           |

스마트폰 사용은 재미, 정보, 사회적 반응이라는 보상을 제공한다. 반복 사용이 장기화되면 보상회로가 강화되고, 스마트폰을 자동적으로 찾게 될 가능성이 높아진다.

---

## 12.2 전전두피질 기능 저하와 부적응

스마트폰 중독이 심해지면 자기조절과 판단을 담당하는 전전두피질의 기능이 약화될 수 있다. 이 경우 다음과 같은 문제가 나타날 수 있다.

| 문제       | 설명                  |
| -------- | ------------------- |
| 충동조절 실패  | 스마트폰 사용을 멈추기 어려움    |
| 학습부진     | 주의집중과 계획 기능 저하      |
| 우울, 불안   | 정서 조절 어려움           |
| 적대감, 공격성 | 편도체 활성 증가와 억제 기능 약화 |

이 관점에서 스마트폰 중독은 단순한 습관 문제가 아니라, 보상회로와 자기조절 기능의 문제로도 이해된다.

# 13. 힐빙-힐텍 접근

11-1 논문은 스마트폰 중독의 예방과 치유를 위해 **힐빙-힐텍 접근**을 제안한다. 힐빙은 웰빙과 힐링을 결합한 개념이고, 힐텍은 이를 실현하기 위한 기술과 방법을 의미한다. 

| 개념       | 의미                  |
| -------- | ------------------- |
| 힐빙       | 몸과 마음의 균형 회복        |
| 힐텍       | 힐빙을 위한 기술과 방법       |
| 힐빙-힐텍 접근 | 자연친화적·통합적 환경을 통한 회복 |

스마트폰 중독자는 스마트폰 중심의 단순한 환경에 갇히기 쉽다. 따라서 자연친화적이고 다양한 환경 자극을 제공하여 전전두피질 기능 회복을 도울 필요가 있다.

예시는 다음과 같다.

| 프로그램      | 효과             |
| --------- | -------------- |
| 산림욕, 숲 체험 | 스트레스 감소, 정서 안정 |

| 미술치료      | 심리적 안정과 표현     |
| 도예치료      | 감각·운동 경험 제공    |
| 댄스치료      | 신체 활동과 정서 조절   |
| 교실 식물 배치  | 자연친화적 환경 조성    |

# 14. VR 기능성게임과 행동주의 적용

11-2 논문은 자폐성 장애 아동의 실생활적응기술 훈련을 위해 VR 기능성게임을 활용하는 방안을 제안한다. 이 논문은 행동주의 원리가 교육용 게임 설계에 어떻게 적용될 수 있는지를 보여준다. 

## 14.1 자폐성 장애 아동에게 VR이 유용한 이유

자폐성 장애 아동은 사회적 의사소통, 상호작용, 일반화, 감각처리에서 어려움을 겪는다. VR은 이들에게 안전하고 예측 가능한 환경을 제공하면서 반복 연습을 가능하게 한다.

| VR의 장점     | 효과           |
| ---------- | ------------ |
| 예측 가능한 환경  | 불안 감소        |
| 감각정보 단순화   | 과도한 자극 부담 완화 |
| 현실과 유사한 상황 | 일반화 촉진       |
| 시각적 정보 제공  | 시각 처리 강점 활용  |
| 반복 훈련 가능   | 실생활 기술 습득 지원 |

---

## 14.2 VR 기능성게임 디자인 프레임워크

논문은 자폐성 장애 아동을 위한 VR 기능성게임 개발 프레임워크를 세 가지로 제시한다.

| 평가              | 사용성 평가, 게임 학습 분석           |

VR 기능성게임은 단순히 재미있는 게임이 아니라, 명확한 학습목표와 평가 체계를 갖춘 교육 도구여야 한다.

이 내용은 행동주의 원리가 단순한 이론이 아니라, 실제 교육 프로그램과 디지털 콘텐츠 설계에도 적용될 수 있음을 보여준다.

# 15. 행동주의 관점에서 성격 형성 예시

행동주의는 성격을 반복적으로 강화된 행동 패턴으로 본다. 자료와 강의에서 제시된 사례를 통합하면 다음과 같다.

| 사례                | 행동 패턴         | 행동주의적 설명                   |
| ----------------- | ------------- | -------------------------- |
| 논쟁하기를 좋아하는 사람     | 자주 반박하고 토론함   | 논쟁 행동이 긍정적 피드백을 받아 강화됨     |
| 자기주장을 하지 않는 사람    | 의견 표현을 피함     | 자기주장 후 부정적 반응을 경험해 행동 감소   |
| 과제 앞에서 아프다고 하는 사람 | 힘든 일을 회피함     | 아프다고 하면 과제를 피할 수 있어 부적 강화됨 |
| 상냥한 사람            | 미소와 친절한 말투 사용 | 타인의 호의와 칭찬을 받아 정적 강화됨      |
| 외향적인 사람           | 사람들과 자주 어울림   | 사회적 상황에서 보상을 많이 경험함        |
| 내향적인 사람           | 조용한 상황 선호     | 혼자 있을 때 안락함을 보상으로 경험함      |

핵심은 “그 사람은 원래 그런 사람이다”가 아니라, **그 행동이 어떤 결과를 경험하며 유지되었는가**이다.

# 16. 훈육과 행동주의

강의에서는 행동주의 원리가 훈육에도 적용된다고 설명했다. 말로만 통제하는 것은 효과가 약하고, 행동 뒤에 실제 결과가 따라와야 한다. 

예를 들어 아이가 마트에서 장난감을 사달라고 떼를 쓸 때, 부모가 어떤 때는 사주고 어떤 때는 안 사주면 문제가 커질 수 있다. 아이는 “떼를 쓰면 언젠가는 원하는 것을 얻을 수 있다”고 학습한다. 이것은 변동비율 강화가 되어 떼쓰기 행동을 더 강하게 유지시킨다.

| 부모 반응     | 아이가 학습하는 것          |
| --------- | ------------------- |
| 매번 사줌     | 떼쓰면 장난감을 얻음         |
| 절대 사주지 않음 | 떼써도 효과 없음           |
| 가끔 사줌     | 오래 떼쓰면 언젠가는 얻을 수 있음 |

가장 위험한 것은 “가끔 사주는 것”이다. 변동비율 강화가 되기 때문에 행동이 쉽게 사라지지 않는다. 따라서 훈육에서는 **일관성**이 중요하다.

# 17. 11주차 핵심 비교: 고전적 조건형성 vs 조작적 조건형성

# 19. 시험 대비 핵심 포인트

## 19.1 반드시 구분해야 할 것

고전적 조건형성과 조작적 조건형성은 반드시 구분해야 한다.

고전적 조건형성은 **자극과 자극**의 연합이다. 예를 들어 치과와 통증이 연결되어 치과만 봐도 불안해지는 것이다.

조작적 조건형성은 **행동과 결과**의 연합이다. 예를 들어 질문했더니 칭찬을 받아 질문 행동이 늘어나는 것이다.

---

## 19.2 강화와 처벌 구분

강화는 행동을 늘리고, 처벌은 행동을 줄인다.
정적은 자극을 주는 것이고, 부적은 자극을 제거하는 것이다.

| 유형    | 판단 기준           |
| ----- | --------------- |
| 정적 강화 | 좋은 것을 줘서 행동 증가  |
| 부적 강화 | 싫은 것을 없애서 행동 증가 |
| 정적 처벌 | 싫은 것을 줘서 행동 감소  |
| 부적 처벌 | 좋은 것을 뺏어서 행동 감소 |

---

## 19.3 강화계획 구분

| 유형    | 키워드            |
| ----- | -------------- |
| 고정비율  | 몇 번 하면 보상      |
| 변동비율  | 몇 번 해야 보상인지 모름 |
| 고정간격  | 일정 시간이 지나면 보상  |
| 변동간격  | 언제 보상인지 모름     |
| 연속 강화 | 매번 보상          |

특히 변동비율 강화는 도박, 랜덤 뽑기, SNS, 스마트폰 중독과 연결된다.

# 20. 최종 요약

11주차는 성격을 행동주의 관점에서 이해하는 내용이다. 행동주의는 성격을 내면의 고정된 본질이 아니라, 환경 속에서 학습된 행동 패턴으로 본다. 인간은 자극에 반응하고, 행동의 결과에 따라 그 행동을 반복하거나 줄인다.

고전적 조건형성은 자극과 자극의 연합을 통해 특정 정서 반응이 학습되는 과정이다. 치과 공포, 특정 장소에 대한 불안, 광고에 대한 호감, 특정 노래에 대한 감정 반응이 이에 해당한다.

조작적 조건형성은 행동과 결과의 연합을 통해 행동이 증가하거나 감소하는 과정이다. 강화는 행동을 증가시키고, 처벌은 행동을 감소시킨다. 행동조성은 원하는 행동을 단계적으로 만들어 가는 방법이며, 강화계획은 행동을 유지시키는 핵심 원리이다.

스마트폰 중독은 변동비율 강화와 변동간격 강화의 구조로 설명할 수 있다. 메시지, 좋아요, 댓글, 쇼츠 같은 보상이 예측 불가능하게 주어지기 때문에 사용자는 계속 스마트폰을 확인한다.

VR 기능성게임 논문은 행동주의 원리가 특수교육과 디지털 훈련 프로그램에도 적용될 수 있음을 보여준다. 자폐성 장애 아동의 실생활적응기술 훈련에는 강화, 처벌, 강화계획, 행동연쇄, 자극촉진, 반응촉진이 중요하다.

결국 11주차의 핵심은 다음 한 문장으로 정리할 수 있다.

> 행동주의에서 성격은 타고난 본질이 아니라, 자극과 반응, 행동과 결과의 반복 속에서 학습되고 강화된 행동 패턴이다.
