# 스마트폰 중독에 대한 행동주의적 분석과 행동신경과학적 힐빙-힐텍 분석 정리

## 1. 논문 개요

**논문 제목**
「스마트폰 중독에 대한 행동주의적 분석과 행동신경과학적인 힐빙-힐텍적 분석을 통한 전인적 교양교육의 필요성 제고」

**저자**
신맹식

**핵심 주제**
이 논문은 스마트폰 중독을 단순한 생활 습관 문제가 아니라 **행동중독**의 하나로 보고, 이를 두 가지 관점에서 분석한다. 첫째는 **행동주의적 관점**, 둘째는 **행동신경과학적 힐빙-힐텍 관점**이다. 최종적으로는 대학생, 특히 대학 초년생을 대상으로 한 **전인적 교양교육** 안에서 스마트폰 중독 예방과 치유 교육이 필요하다고 주장한다. 

---

## 2. 연구의 문제의식

현대사회에서 스마트폰은 정보 수집, 의사소통, SNS, 여가 활동 등을 가능하게 하는 필수 도구가 되었다. 하지만 그 이면에는 **강박적 사용**과 **스마트폰 중독**이라는 문제가 존재한다.

논문은 특히 청소년과 대학생 초년생에게 스마트폰 중독 비율이 높다는 점에 주목한다. 이 시기는 뇌, 신체, 성격, 인지, 도덕성 등이 성숙해지는 중요한 시기이므로, 스마트폰 중독은 전인적 성장과 대학 교양교육의 목표 달성에 방해가 될 수 있다.

즉, 논문의 핵심 문제의식은 다음과 같다.

> 스마트폰 중독은 청소년과 대학생의 전인적 인성 형성에 부정적 영향을 미치며, 대학 교양교육 차원에서 예방과 치유가 필요하다.

---

# 3. 스마트폰 중독의 의미

## 3.1 스마트폰 중독이란?

스마트폰 중독은 스마트폰에 과도하게 몰입한 결과, 스스로 사용을 통제하기 어려운 상태를 말한다.

논문에서는 스마트폰 중독을 **행동중독**의 한 유형으로 본다. 행동중독은 약물중독처럼 특정 물질을 섭취하는 것은 아니지만, 특정 행동을 반복적으로 통제하지 못하는 상태를 의미한다.

대표적인 행동중독에는 다음이 있다.

| 행동중독 유형 | 설명                 |
| ------- | ------------------ |
| 병적 도박   | 도박 행동을 통제하지 못함     |
| 병적 도벽   | 훔치는 행동을 반복함        |
| 강박적 구매  | 필요 이상으로 구매 행동을 반복함 |
| 인터넷 중독  | 인터넷 사용을 조절하지 못함    |
| 스마트폰 중독 | 스마트폰 사용을 통제하지 못함   |

---

## 3.2 스마트폰 중독의 주요 증상

논문은 스마트폰 중독이 약물중독과 유사한 증상을 보인다고 설명한다.

| 증상      | 의미                         |
| ------- | -------------------------- |
| 갈망      | 스마트폰을 계속 확인하고 싶은 강한 욕구     |
| 내성      | 점점 더 오래 사용해야 만족을 느낌        |
| 금단      | 스마트폰을 사용하지 못하면 불안하거나 초조함   |
| 일상생활 장애 | 학업, 대인관계, 수면, 생활 리듬에 문제 발생 |

스마트폰 중독은 단순히 “많이 쓰는 것”이 아니라, **사용을 조절하지 못하고 생활 기능이 손상되는 상태**라는 점이 중요하다.

# 4. 스마트폰 사용의 주요 동기

논문은 사람들이 스마트폰을 사용하는 주요 이유를 다음과 같이 정리한다.

| 사용 동기  | 내용                                 |
| ------ | ---------------------------------- |
| 유희성    | 영상 시청, 음악 감상, 게임, 앱 사용 등 재미와 흥미 추구 |
| 사교성    | 카카오톡, 페이스북, SNS 등을 통한 인간관계 유지      |
| 유용성    | 정보 검색, 일정 관리, 학습, 업무 등 실생활 도움      |
| 혁신성    | 새로운 기술을 사용해보고자 하는 태도               |
| 주관적 규범 | 스마트폰 사용이 사회적으로 당연하다는 압력            |

특히 스마트폰은 **재미**, **관계**, **정보**를 즉각적으로 제공하기 때문에 사용자가 쉽게 반복 사용에 빠질 수 있다.

# 5. 행동주의적 관점에서 본 스마트폰 중독

## 5.1 행동주의의 기본 관점

행동주의는 인간 행동을 내면의 정신 과정이 아니라 **관찰 가능한 자극과 반응의 관계**로 설명한다. 특히 어떤 행동이 반복되는 이유를 **강화**의 원리로 설명한다.

스마트폰 중독을 행동주의적으로 보면 다음과 같다.

> 스마트폰 사용 행동은 스마트폰이 제공하는 보상 때문에 반복되고, 반복된 행동이 습관화되면서 중독으로 발전한다.

---

## 5.2 정적 강화와 스마트폰 사용

정적 강화는 어떤 행동 뒤에 긍정적 자극이 제공되어 그 행동이 증가하는 것을 말한다.

스마트폰 사용의 경우, 사용자는 스마트폰을 확인한 뒤 여러 보상을 얻는다.

| 스마트폰 사용 행동 | 뒤따르는 보상         |
| ---------- | --------------- |
| 카카오톡 확인    | 친구의 메시지         |
| SNS 확인     | 좋아요, 댓글, 새로운 소식 |
| 유튜브 시청     | 재미, 지루함 해소      |
| 게임 실행      | 흥미, 성취감         |
| 검색         | 필요한 정보 획득       |

따라서 스마트폰 사용은 **정적 강화**에 의해 유지된다. 스마트폰을 확인할 때마다 재미, 정보, 인간관계 반응 같은 보상이 주어지기 때문에 사용자는 계속 스마트폰을 보게 된다.

---

## 5.3 가변 강화계획과 스마트폰 중독

논문에서 가장 중요한 행동주의 개념은 **가변 강화계획**이다.

가변 강화계획은 보상이 언제 주어질지 예측하기 어려운 방식이다. 예를 들어 도박에서 슬롯머신을 몇 번 당겨야 보상이 나올지 알 수 없는 것과 같다. 이 방식은 행동을 매우 강하게 유지시킨다.

스마트폰도 비슷하다.

| 행동       | 보상 여부                       |
| -------- | --------------------------- |
| 카카오톡 확인  | 메시지가 있을 수도 있고 없을 수도 있음      |
| 인스타그램 확인 | 흥미로운 게시물이 있을 수도 있고 없을 수도 있음 |
| 이메일 확인   | 중요한 메일이 있을 수도 있고 없을 수도 있음   |
| 알림 확인    | 유용한 정보일 수도 있고 아닐 수도 있음      |

즉, 스마트폰 사용자는 매번 보상을 받는 것이 아니라 **가끔 예측 불가능하게 보상**을 받는다. 이 때문에 사용자는 “혹시 뭔가 왔을까?”라는 기대감으로 계속 스마트폰을 확인하게 된다.

이 구조는 도박 중독과 유사하다.

---

## 5.4 가변비율 강화계획과 가변간격 강화계획

논문은 스마트폰 중독을 특히 **가변비율 강화계획**과 **가변간격 강화계획**으로 설명한다.

| 강화계획      | 의미                  | 스마트폰 예시                          |
| --------- | ------------------- | -------------------------------- |
| 가변비율 강화계획 | 몇 번 행동해야 보상이 나올지 모름 | 몇 번 SNS를 새로고침해야 흥미로운 게시물이 나올지 모름 |
| 가변간격 강화계획 | 어느 시간 후 보상이 나올지 모름  | 언제 메시지나 알림이 올지 모름                |

스마트폰은 이 두 가지 강화계획이 결합된 매체이다. 그래서 사용자는 특정 시간이나 장소와 관계없이 계속 스마트폰을 확인하게 된다.

---

# 6. 행동주의적 해결 방안

## 6.1 가변 강화계획에서 고정 강화계획으로 전환

논문은 스마트폰 중독을 줄이기 위해 **가변 강화계획을 고정 강화계획으로 바꾸어야 한다**고 제안한다.

즉, 스마트폰을 아무 때나 확인하는 것이 아니라 **정해진 시간**, **정해진 횟수**에만 사용하도록 제한하는 것이다.

예를 들어 다음과 같은 방식이다.

| 방식        | 예시                   |
| --------- | -------------------- |
| 고정간격 강화계획 | 4시간마다 한 번 스마트폰 사용 허용 |
| 고정비율 강화계획 | 하루 4번만 스마트폰 확인 허용    |
| 시간 제한     | 한 번 사용할 때 최대 30분만 허용 |

이 방법의 핵심은 스마트폰 사용을 예측 가능한 구조로 바꾸는 것이다. 사용자가 아무 때나 스마트폰을 확인하지 않도록 만들면, 스마트폰 사용 빈도와 몰입 시간이 줄어들 수 있다.

---

## 6.2 대학 교양교육에서의 적용

논문은 대학 초년생을 대상으로 스마트폰 사용 습관을 점검하고, 필요한 경우 사용 조절 훈련을 교양교육 프로그램에 포함해야 한다고 주장한다.

예를 들어 다음과 같은 교육이 가능하다.

| 교육 내용        | 설명                       |
| ------------ | ------------------------ |
| 스마트폰 중독 검사   | 학생의 스마트폰 의존 정도 파악        |
| 사용 시간 기록     | 하루 사용 시간과 사용 상황 점검       |
| 고정 사용 계획 세우기 | 정해진 시간과 횟수에만 사용          |
| 대체 행동 훈련     | 운동, 독서, 대화, 자연 체험 등으로 대체 |
| 자기조절 교육      | 충동 조절과 시간 관리 훈련          |

# 7. 행동신경과학적 관점에서 본 스마트폰 중독

## 7.1 행동주의 관점의 한계

행동주의는 스마트폰 중독을 외현적 행동과 보상 구조로 설명하는 데 유용하다. 하지만 우울, 불안, 충동조절 실패, 학습능률 저하 같은 **내적 심리·신경 과정**을 설명하는 데에는 한계가 있다.

그래서 논문은 행동신경과학적 관점을 추가한다.

---

## 7.2 스마트폰 중독과 뇌 회로

논문은 스마트폰 중독이 뇌의 보상회로와 관련된다고 설명한다. 핵심 뇌 영역은 다음과 같다.

| 뇌 영역       | 기능                  |
| ---------- | ------------------- |
| 전전두피질, PFC | 자기조절, 판단, 계획, 충동 억제 |
| 복측피개야, VTA | 도파민 보상회로의 출발점       |
| 측핵         | 보상, 쾌감, 강화 학습       |
| 편도체        | 공포, 불안, 공격성, 정서 반응  |
| 변연계        | 정서와 동기 조절           |

스마트폰을 사용하면 재미, 정보, 사회적 반응이라는 보상이 주어진다. 이때 도파민 보상회로가 활성화된다. 반복 사용이 장기화되면 스마트폰 사용 행동과 보상 사이의 연결이 강해지고, 사용자는 점점 더 스마트폰을 자동적으로 찾게 된다.

---

## 7.3 전전두피질 기능 저하

논문은 스마트폰 중독이 심해지면 **전전두피질(PFC)**의 기능이 약화될 수 있다고 본다.

전전두피질은 다음 기능을 담당한다.

| 기능    | 설명               |
| ----- | ---------------- |
| 자기조절  | 하고 싶은 행동을 참는 능력  |
| 판단    | 무엇이 필요한지 결정하는 능력 |
| 계획    | 목표를 세우고 실행하는 능력  |
| 주의집중  | 학습이나 과제에 집중하는 능력 |
| 충동 억제 | 즉각적 욕구를 조절하는 능력  |

전전두피질 기능이 약화되면 스마트폰 사용을 멈추기 어렵고, 학업 집중이 떨어지며, 충동적 행동이나 부정적 정서가 증가할 수 있다.

---

## 7.4 스마트폰 중독과 부적응 증상

논문은 스마트폰 중독이 다음과 같은 문제와 관련될 수 있다고 설명한다.

| 증상       | 행동신경과학적 설명              |
| -------- | ----------------------- |
| 우울       | 정서 조절 기능 약화             |
| 불안       | 편도체 활성 증가               |
| 적대감, 공격성 | 충동 억제 기능 저하             |
| 학습부진     | 전전두피질 기능 저하로 주의집중 약화    |
| 자기조절 실패  | 보상회로 과활성화와 PFC 억제 기능 약화 |

즉, 스마트폰 중독은 단순히 시간을 많이 빼앗는 문제가 아니라, **인지 기능과 정서 조절 능력에도 영향을 줄 수 있는 문제**로 이해된다.

# 8. 힐빙-힐텍적 관점

## 8.1 힐빙과 힐텍의 의미

논문에서 말하는 **힐빙**은 웰빙과 힐링이 결합된 개념이다. 신체적·정신적 건강을 균형 있게 회복하는 것을 의미한다.

**힐텍**은 힐빙을 달성하기 위한 기술이나 방법을 뜻한다. 여기에는 자연친화적 환경, 치유 프로그램, 예술, 문화, 생명과학적 접근 등이 포함된다.

정리하면 다음과 같다.

| 개념       | 의미                            |
| -------- | ----------------------------- |
| 힐빙       | 몸과 마음의 균형을 회복하는 상태            |
| 힐텍       | 힐빙을 실현하기 위한 기술과 방법            |
| 힐빙-힐텍 접근 | 자연친화적·통합적 환경을 통해 심신 회복을 돕는 접근 |

---

## 8.2 힐빙-힐텍 접근의 핵심

논문은 스마트폰 중독을 극복하기 위해 단순히 “사용하지 마라”라고 하는 것만으로는 부족하다고 본다.

스마트폰 중독자는 스마트폰 중심의 단순하고 제한된 환경에 갇히기 쉽다. 따라서 이들에게는 스마트폰 외부의 풍부한 자극, 특히 자연친화적이고 통합적인 환경 경험이 필요하다.

예시로는 다음이 있다.

| 힐빙-힐텍 프로그램 | 내용               |
| ---------- | ---------------- |
| 산림욕        | 숲에서 걷기, 자연 바라보기  |
| 숲 체험       | 자연환경 속에서 감각 경험하기 |
| 음악치료       | 음악을 통한 정서 회복     |
| 미술치료       | 표현 활동을 통한 심리 안정  |
| 도예치료       | 손을 사용하는 창작 활동    |
| 댄스치료       | 신체 활동을 통한 정서 조절  |
| 차 치료       | 차를 마시며 안정감 회복    |
| 교실 내 식물 배치 | 학습 환경에 자연 요소 도입  |

이러한 활동은 전전두피질의 기능 회복, 정서 안정, 스트레스 감소, 자기조절 능력 향상에 도움이 될 수 있다고 논문은 설명한다.

# 9. 논문의 핵심 주장

논문의 핵심 주장은 다음과 같이 정리할 수 있다.

## 9.1 스마트폰 중독은 행동주의적으로 설명 가능하다

스마트폰은 예측 불가능한 보상을 제공한다. 메시지, 알림, 댓글, 좋아요, 흥미로운 콘텐츠가 언제 나타날지 모르기 때문에 사용자는 계속 확인한다. 이는 **가변 강화계획**의 원리와 유사하다.

## 9.2 스마트폰 중독은 뇌 기능과도 관련된다

반복적인 스마트폰 사용은 보상회로를 강화하고, 전전두피질의 자기조절 기능을 약화시킬 수 있다. 그 결과 충동조절 실패, 학습부진, 우울, 불안, 적대감 등이 나타날 수 있다.

## 9.3 해결을 위해서는 행동 조절과 환경 변화가 함께 필요하다

스마트폰 사용을 줄이기 위해서는 사용 시간과 횟수를 정하는 **고정 강화계획**이 필요하다. 또한 이미 중독이 심한 경우에는 자연친화적이고 풍부한 환경을 제공하여 전전두피질 기능 회복을 도와야 한다.

## 9.4 대학 교양교육에서 다루어야 한다

대학생 초년생은 전인적 인성 형성의 중요한 시기에 있다. 따라서 대학은 스마트폰 중독 예방과 치유를 교양교육의 일부로 다룰 필요가 있다.

# 10. 논문의 의의

이 논문의 의의는 스마트폰 중독을 단순한 사회문제나 생활습관 문제가 아니라, **행동주의**, **강화계획**, **보상회로**, **전전두피질**, **힐빙-힐텍**이라는 다층적 관점에서 설명했다는 점이다.

특히 기존 연구들이 주로 스마트폰 중독의 현상이나 증상에 초점을 맞췄다면, 이 논문은 다음 두 가지를 강조한다.

첫째, 스마트폰 중독은 가변 강화계획에 의해 유지되는 행동중독이다.
둘째, 스마트폰 중독이 심화되면 전전두피질의 자기조절 기능 약화와 관련될 수 있으므로, 자연친화적·통합적 환경을 통한 회복이 필요하다.

# 11. 논문의 한계

논문은 이론적 리뷰 연구이기 때문에, 제안한 프로그램의 효과가 실제로 검증된 것은 아니다.

따라서 한계는 다음과 같다.

| 한계            | 설명                                      |
| ------------- | --------------------------------------- |
| 이론 중심 연구      | 실제 실험이나 프로그램 효과 검증이 부족함                 |
| 구체적 프로그램 부족   | 교양교육에서 어떻게 운영할지 세부 설계가 더 필요함            |
| 스마트폰 중독 특성 변화 | 논문 작성 이후 스마트폰 사용 환경이 크게 변화했을 가능성 있음     |
| 개인차 고려 부족     | 개인의 성격, 환경, 정신건강 수준에 따른 차이를 더 고려할 필요 있음 |

# 13. 최종 요약

이 논문은 스마트폰 중독을 행동주의와 행동신경과학의 관점에서 분석한다. 행동주의적으로 보면 스마트폰은 메시지, 알림, SNS 반응, 재미있는 콘텐츠와 같은 보상을 예측 불가능하게 제공하기 때문에 사용자를 계속 확인하게 만든다. 이는 도박과 유사한 **가변 강화계획**의 구조를 가진다.

해결 방안으로는 스마트폰을 아무 때나 사용하는 습관을 줄이고, 정해진 시간과 횟수에만 사용하는 **고정 강화계획**을 적용할 필요가 있다.

또한 스마트폰 중독이 심해지면 자기조절과 판단을 담당하는 **전전두피질**의 기능이 약화되고, 보상회로와 정서회로가 과도하게 작동할 수 있다. 이로 인해 우울, 불안, 충동조절 실패, 학습부진이 나타날 수 있다.

따라서 논문은 스마트폰 중독을 예방하고 치유하기 위해 대학 교양교육에서 스마트폰 사용 조절 훈련과 함께 산림욕, 숲 체험, 음악치료, 미술치료 등 자연친화적이고 통합적인 **힐빙-힐텍 프로그램**을 운영할 필요가 있다고 주장한다.
