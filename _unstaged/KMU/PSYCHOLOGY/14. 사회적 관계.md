# 14. 사회적 관계

## 1. 14주차 전체 주제

14주차의 핵심 주제는 **사회적 관계 속에서 개인이 어떻게 달라지는가**이다.

사회심리학은 개인의 행동을 단순히 “그 사람이 원래 그런 사람이라서”라고 설명하지 않는다. 오히려 **타인의 존재, 집단 구조, 평가 가능성, 책임감 분산, 익명성, 역할, 집단 의사결정 과정** 같은 상황 요인이 개인의 행동을 크게 바꾼다고 본다.

이번 주차에서 다루는 주요 개념은 다음과 같다.

| 개념     | 핵심 의미                           |
| ------ | ------------------------------- |
| 사회적 촉진 | 타인의 존재로 개인 수행이 향상되는 현상          |
| 사회적 억제 | 타인의 존재로 개인 수행이 저하되는 현상          |
| 우세반응   | 각성 상태에서 가장 익숙하고 자동화된 반응이 강화되는 것 |
| 사회적 태만 | 집단 속에서 개인의 노력이 감소하는 현상          |
| 책임감 분산 | 사람이 많을수록 개인 책임감이 약해지는 현상        |
| 방관자 효과 | 주변 사람이 많을수록 도움 행동이 감소하는 현상      |
| 몰개성화   | 군중 속에서 개인 정체성과 책임감이 약화되는 현상     |
| 사회적 역할 | 개인이 맡은 역할이 행동과 정체성에 영향을 주는 것    |
| 집단사고   | 집단의 합의 욕구 때문에 비판적 사고가 억제되는 현상   |
| 집단 극화  | 집단 토론 후 의견이 더 극단화되는 현상          |

## 2. 사회적 촉진

### 2.1 개념

**사회적 촉진 social facilitation**은 타인이 있을 때 개인의 수행 수준이 높아지는 현상이다.

혼자 과제를 수행할 때보다 다른 사람과 함께 있거나, 누군가가 지켜보는 상황에서 수행이 좋아지는 경우를 말한다.

예를 들어 다음과 같은 상황이 있다.

| 예시      | 설명                                   |
| ------- | ------------------------------------ |
| 자전거 경기  | 혼자 탈 때보다 다른 사람과 함께 탈 때 속도가 빨라짐       |
| 도서관 공부  | 주변에 공부하는 사람이 있으면 나도 더 집중하게 됨         |
| 스터디 위드미 | 실제로 함께 있지 않아도 타인의 존재감을 통해 공부 지속이 촉진됨 |
| 헬스장 운동  | 집보다 헬스장에서 운동할 때 수행이 좋아질 수 있음         |
| 공연장 무대  | 관객의 반응이 공연자의 수행을 끌어올릴 수 있음           |

녹음에서는 코로나 시기 무관중 공연 사례가 언급되었다. 관객 앞에서는 실수하지 않던 공연자가 조용한 공간에서 카메라만 보고 공연할 때 오히려 실수를 많이 했다는 것이다. 이는 관객의 존재가 공연자에게 사회적 촉진을 일으켰음을 보여준다. 

### 2.2 Triplett의 연구

사회적 촉진의 대표적 초기 연구는 **Triplett**의 자전거 연구이다.

Triplett은 자전거 선수들이 혼자 달릴 때보다 다른 사람과 함께 달릴 때 더 빠른 기록을 낸다는 점을 발견했다. 또한 아동들이 낚싯줄을 감는 과제에서도 혼자 할 때보다 함께 있을 때 더 빠르게 수행하는 경향이 나타났다.

중요한 점은 경쟁 때문만이 아니라는 것이다.
단순히 **타인의 존재**나 **공동 행동** 자체가 수행을 높일 수 있다.

## 3. 사회적 촉진은 항상 수행을 높이는가?

사회적 촉진은 “타인이 있으면 무조건 잘한다”는 뜻이 아니다.

타인의 존재는 때로 수행을 향상시키지만, 때로는 오히려 수행을 방해한다. 발표, 오디션, 시험, 무대 상황에서 연습 때보다 실수를 많이 하는 경우가 여기에 해당한다.

이를 설명하기 위해 등장하는 개념이 **사회적 억제 social inhibition**이다.

| 구분     | 의미              |
| ------ | --------------- |
| 사회적 촉진 | 타인의 존재로 수행이 향상됨 |
| 사회적 억제 | 타인의 존재로 수행이 저하됨 |

이 차이를 설명하는 핵심 이론이 **Zajonc의 우세반응 이론**이다.

## 4. Zajonc의 우세반응 이론

### 4.1 핵심 원리

Zajonc는 타인의 존재가 과제 수행 자체를 무조건 높이는 것이 아니라, 개인의 **우세반응 dominant response**을 활성화한다고 보았다.

**우세반응**이란 어떤 상황에서 가장 익숙하고 자동적으로 나오는 반응이다.

타인의 존재는 각성을 증가시키고, 이 각성은 우세반응을 강화한다.

```text
타인의 존재
→ 각성 증가
→ 우세반응 활성화
→ 과제 특성에 따라 수행 향상 또는 수행 저하
```

### 4.2 과제 난이도와 숙련도

우세반응 이론에서 중요한 기준은 **과제 난이도**와 **숙련도**이다.

| 과제 상태     | 우세반응            | 결과    |
| --------- | --------------- | ----- |
| 쉬운 과제     | 정답 반응이 우세       | 수행 향상 |
| 익숙한 과제    | 숙련된 반응이 우세      | 수행 향상 |
| 어려운 과제    | 오답 반응이 우세할 수 있음 | 수행 저하 |
| 처음 배우는 과제 | 미숙한 반응이 우세      | 수행 저하 |

즉, 잘 연습된 과제에서는 긴장과 각성이 수행을 높일 수 있다. 반대로 아직 익숙하지 않은 과제에서는 긴장과 각성이 실수를 증가시킬 수 있다.

녹음에서는 윤여정 배우의 사례가 언급되었다. 숙련된 배우는 카메라 앞에서 긴장하더라도 오랜 연습으로 만들어진 우세반응이 자동적으로 나오기 때문에 수행이 촉진될 수 있다는 설명이었다. 

## 5. 사회적 촉진을 설명하는 이론들

### 5.1 활성화 이론

활성화 이론은 타인의 존재가 생리적 각성을 증가시킨다고 본다.

각성이 증가하면 우세반응이 강화된다. 따라서 단순하고 익숙한 과제에서는 수행이 향상되고, 복잡하고 어려운 과제에서는 수행이 저하될 수 있다.

### 5.2 평가우려 이론

평가우려 이론은 타인의 존재 자체보다 **타인이 나를 평가할 수 있다는 인식**이 중요하다고 본다.

누군가가 나의 수행을 평가한다고 느끼면 긴장과 각성이 증가한다. 이 각성은 수행에 영향을 미친다.

### 5.3 자기제시 이론

**자기제시 이론 self-presentation theory**은 사람들이 타인에게 좋은 인상을 주고 싶어 한다는 점에 주목한다.

사람은 타인 앞에서 이상적인 자기 모습을 보이고자 한다. 그래서 타인이 있을 때 더 노력하고, 더 잘하려는 동기가 생긴다.

### 5.4 자의식 이론

**자의식 이론 self-awareness theory**은 타인의 존재가 자신에 대한 주의집중을 높인다고 본다.

타인이 있으면 “내가 어떻게 보일까?”를 더 의식하게 된다. 이 자의식이 각성을 높이고, 그 결과 수행에 영향을 준다.

### 5.5 주의분산-갈등 이론

**주의분산-갈등 이론 distraction-conflict theory**은 타인의 존재가 주의를 분산시키기 때문에 각성이 증가한다고 본다.

타인이 있을 때 사람은 두 가지 요구 사이에서 갈등한다.

| 주의 방향  | 설명                     |
| ------ | ---------------------- |
| 과제에 집중 | 수행을 잘하기 위해 과제에 몰입해야 함  |
| 타인을 의식 | 타인이 나를 어떻게 보는지 신경 쓰게 됨 |

이 갈등이 각성을 증가시키고, 결과적으로 수행을 촉진하거나 억제한다.

## 6. 사회적 태만

### 6.1 개념

**사회적 태만 social loafing**은 여러 사람이 함께 과제를 수행할 때 개인의 노력이 줄어드는 현상이다.

사회적 촉진이 타인의 존재로 수행이 올라가는 현상이라면, 사회적 태만은 집단 속에서 개인의 수행이 낮아지는 현상이다.

쉽게 말해, 사회적 태만은 **무임승차**와 연결된다.

### 6.2 Ringelmann 효과

사회적 태만의 대표적 근거는 **Ringelmann 효과**이다.

Ringelmann은 사람들이 줄다리기를 할 때 혼자 할 때보다 집단으로 할 때 개인당 힘이 줄어든다는 것을 발견했다.

녹음에서는 다음 수치가 언급되었다.

| 조건    | 개인당 평균 압력 |
| ----- | --------- |
| 혼자 수행 | 약 63kg    |
| 3명 집단 | 약 53kg    |
| 8명 집단 | 약 31kg    |

즉, 집단 규모가 커질수록 전체 힘은 증가할 수 있지만, 개인 한 명당 투입하는 평균 노력은 감소한다. 

### 6.3 사회적 태만의 원인

사회적 태만은 개인이 원래 게을러서만 생기는 것이 아니다. 집단 구조상 개인의 기여와 책임이 흐려질 때 발생한다.

| 원인            | 설명                   |
| ------------- | -------------------- |
| 개인 기여도 확인 불가능 | 내가 얼마나 했는지 잘 드러나지 않음 |
| 평가 불명확성       | 개인별 수행 정도를 평가하기 어려움  |
| 책임감 분산        | 누군가 하겠지라고 생각함        |
| 역할 분담 불명확     | 누가 무엇을 해야 하는지 모호함    |
| 의사소통 문제       | 팀원 간 조율이 어려움         |
| 리더십 부족        | 전체 노력이 효과적으로 통합되지 않음 |

팀플을 싫어하는 이유도 이와 연결된다. 학생들이 팀 과제를 부담스럽게 느끼는 이유 중 하나는 연락과 조율의 어려움뿐 아니라, **노력하지 않는 팀원**을 경험하기 때문이다.

## 7. 사회적 태만을 줄이는 방법

사회적 태만을 줄이기 위해서는 개인의 책임과 기여를 명확히 해야 한다.

| 방법         | 설명                         |
| ---------- | -------------------------- |
| 개인 기여도 평가  | 누가 얼마나 기여했는지 확인 가능하게 함     |
| 명확한 역할 분담  | 각자의 책임 범위를 분명히 함           |
| 개인 동기 유발   | 개인의 노력이 집단 성공에 중요하다는 인식 제공 |
| 수행 감찰      | 개인 수행을 관찰하고 피드백 제공         |
| 노력 수준 공표   | 자신이 투입할 노력 수준을 공개하게 함      |
| 과제 흥미 부여   | 과제를 의미 있거나 흥미롭게 만듦         |
| 보상 제공      | 참여에 대한 인정과 보상 제공           |
| 집단 응집성 강화  | 팀 정체성과 소속감을 높임             |
| 집단 간 경쟁 유도 | 다른 팀과 비교하여 동기 강화           |

녹음에서는 게임 길드 사례가 제시되었다. 잘 운영되는 길드는 접속하지 않는 사람을 관리하고, 개인별 점수를 확인하고, 참여하지 못하는 이유를 공유하게 하며, 보상을 통해 동기를 높인다. 이는 사회적 태만을 줄이는 실제적 장치로 볼 수 있다. 

## 8. 과제 유형과 집단 생산성

집단 수행은 과제 유형에 따라 달라진다. Steiner는 집단 과제를 크게 **가산과제, 결합과제, 분리과제**로 구분했다.

### 8.1 가산과제

**가산과제 additive task**는 구성원 개인의 노력이 합산되어 집단 생산성이 결정되는 과제이다.

예시는 다음과 같다.

| 예시                      |
| ----------------------- |
| 줄다리기                    |
| 운동장 쓰레기 줍기              |
| 트럭 물건 내리기               |
| 여러 사람이 점수를 모으는 게임 길드 활동 |

가산과제에서는 집단 수행이 개인 수행보다 대체로 우수하다. 하지만 개인별 기여도가 불분명하면 사회적 태만이 쉽게 발생한다.

### 8.2 결합과제

**결합과제 conjunctive task**는 모든 구성원이 일정 수준 이상의 수행을 해야 집단 전체가 성공하는 과제이다.

예시는 다음과 같다.

| 예시          |
| ----------- |
| 스파이들의 적진 침투 |
| 조립 라인 작업    |
| 조별 프로젝트     |
| 팀 발표        |

결합과제에서는 가장 낮은 수행을 보이는 구성원이 전체 결과를 좌우할 수 있다. 따라서 역할 분담, 훈련, 구성원 선택, 책임감이 중요하다.

### 8.3 분리과제

**분리과제 disjunctive task**는 집단 구성원 중 한 명만 성공해도 집단 전체가 성공하는 과제이다.

예시는 다음과 같다.

| 예시                   |
| -------------------- |
| 퀴즈 대결                |
| 한 명이 문제를 해결하는 팀 활동   |
| 잘하는 사람이 팀을 “캐리”하는 상황 |

분리과제에서는 가장 유능한 구성원이 집단 수행을 좌우한다. 반대로 다른 구성원들은 유능한 사람에게 의존하며 태만해질 수 있다.

### 8.4 과제 유형 비교

| 과제 유형 | 집단 성공 조건       | 태만 가능성              |
| ----- | -------------- | ------------------- |
| 가산과제  | 개인 노력의 합       | 개인 기여가 묻힐 때 태만 발생   |
| 결합과제  | 모두 일정 수준 이상 수행 | 낮은 수행자가 전체 결과를 제한   |
| 분리과제  | 한 명만 성공해도 됨    | 유능한 사람에게 의존하며 태만 발생 |

## 9. 책임감 분산과 방관자 효과

### 9.1 책임감 분산

**책임감 분산 diffusion of responsibility**은 사람이 많을수록 개인이 느끼는 책임감이 줄어드는 현상이다.

혼자 있을 때는 “내가 해야 한다”고 느끼지만, 여러 사람이 있을 때는 “누군가 하겠지”라고 생각하기 쉽다.

### 9.2 방관자 효과

**방관자 효과 bystander effect**는 주변 사람이 많을수록 도움 행동이 감소하는 현상이다.

사람이 많으면 도움을 줄 가능성이 높아질 것 같지만, 실제로는 책임감이 분산되어 도움 행동이 줄어들 수 있다.

### 9.3 키티 제노비스 사건

수업에서는 **키티 제노비스 사건**이 언급되었다. 언론은 이 사건을 “많은 목격자가 있었지만 아무도 경찰에 신고하지 않았다”는 방식으로 보도했고, 처음에는 도시 사람들의 냉담함과 무관심으로 해석했다.

그러나 사회심리학자들은 이를 개인의 비도덕성으로만 설명하지 않았다.
그들은 많은 사람이 있었기 때문에 각 개인의 책임감이 희석되었을 가능성, 즉 **방관자 효과**에 주목했다.

이 관점이 중요한 이유는 다음과 같다.

| 일반적 해석       | 사회심리학적 해석           |
| ------------ | ------------------- |
| 사람들의 인성이 나쁘다 | 상황이 도움 행동을 어렵게 만들었다 |
| 도시 사람들은 냉담하다 | 타인의 존재가 책임감을 분산시켰다  |
| 도덕성 문제다      | 집단 상황의 심리적 메커니즘 문제다 |

## 10. Latane의 도움 행동 의사결정 모형

Latane와 Darley는 응급상황에서 도움 행동이 자동적으로 발생하는 것이 아니라고 보았다. 도움 행동이 일어나기 위해서는 다섯 단계를 통과해야 한다.

| 단계  | 내용          | 실패 요인                   |
| --- | ----------- | ----------------------- |
| 1단계 | 사건을 알아차림    | 사람이 많거나 시끄러우면 인식이 늦어짐   |
| 2단계 | 응급상황으로 해석   | 상황이 애매하면 판단이 어려움        |
| 3단계 | 책임감을 느낌     | 사람이 많으면 책임감이 분산됨        |
| 4단계 | 도움 방법을 앎    | 무엇을 해야 할지 모를 수 있음       |
| 5단계 | 실제 도움 행동을 함 | 위험, 법적 문제, 시간 비용 등을 걱정함 |

이 중 어느 한 단계에서 실패해도 도움 행동은 일어나지 않을 수 있다.

### 10.1 사회심리학적 의의

이 모형은 사람들이 위기 상황에서 바로 돕지 않는 이유를 개인의 성격이나 도덕성만으로 설명하지 않는다.

오히려 도움 행동이 일어나기까지는 다음과 같은 심리적 장벽이 있음을 보여준다.

| 장벽      | 설명                  |
| ------- | ------------------- |
| 인식의 어려움 | 위기 상황을 놓칠 수 있음      |
| 해석의 어려움 | 위기인지 아닌지 애매할 수 있음   |
| 책임감 분산  | 다른 사람이 할 것이라 생각함    |
| 지식 부족   | 어떻게 도와야 할지 모름       |
| 비용 고려   | 자신에게 생길 위험과 손해를 계산함 |

## 11. AI 위험 모니터링과 도움 행동 모형의 적용

녹음에서는 AI 기반 실시간 위험 모니터링 시스템 사례가 언급되었다.

이 시스템은 CCTV와 AI를 활용해 위험 상황을 더 빠르게 인식하도록 돕는다. Latane의 모형에 적용하면, AI는 주로 1단계인 **사건 인식**을 보완한다.

만약 AI가 상황을 응급상황으로 해석해 준다면 2단계도 보완할 수 있다.

그러나 실제 도움 행동이 일어나려면 3~5단계도 해결되어야 한다.

| 단계       | 시스템 설계상 보완점               |
| -------- | ------------------------- |
| 책임감 부여   | 특정 사람이나 기관에 명확한 요청 전달     |
| 도움 방법 제시 | 신고, 대피, 응급처치 등 구체적 선택지 제공 |
| 행동 비용 감소 | 법적·신체적 위험을 줄이는 절차 안내      |

이 사례는 심리학 이론이 실제 사회문제 해결 시스템 설계에도 적용될 수 있음을 보여준다.

## 12. 몰개성화

### 12.1 개념

**몰개성화 deindividuation**는 집단이나 군중 속에서 개인의 정체감과 책임감이 약화되는 현상이다.

혼자 있을 때는 하지 않을 행동도 군중 속에서는 쉽게 하게 될 수 있다.

몰개성화는 **자아정체망각**, **탈개인화**라고도 불린다.

### 12.2 몰개성화의 조건

몰개성화는 다음 조건에서 쉽게 발생한다.

| 조건         | 설명                         |
| ---------- | -------------------------- |
| 익명성        | 내가 누구인지 드러나지 않음            |
| 책임감 분산     | 행동의 책임이 개인에게 집중되지 않음       |
| 자극 과잉      | 소리, 빛, 흥분, 과도한 인지적 부하      |
| 생리적 흥분     | 감정적으로 고조된 상태               |
| 자기구분 단서 부족 | 군중과 나를 구분하기 어려움            |
| 동일한 복장·상징  | 같은 유니폼, 응원 도구 등으로 개인성이 약화됨 |

예를 들어 스포츠 경기 후 군중 폭력, 온라인 익명 커뮤니티의 공격적 댓글, 집단적 마녀사냥 등이 몰개성화와 연결될 수 있다.

### 12.3 몰개성화의 과정

몰개성화는 다음 흐름으로 이해할 수 있다.

```text
익명성 + 책임감 분산 + 과도한 각성
→ 자의식 감소
→ 평가불안 감소
→ 자기조절 약화
→ 수치심·죄책감 약화
→ 충동적·공격적 행동 증가
```

사회적 촉진에서는 타인의 존재가 자의식과 평가불안을 높이는 반면, 몰개성화에서는 익명성과 군중 속 동일화로 인해 자의식과 평가불안이 낮아진다.

## 13. 스탠포드 감옥 실험

### 13.1 실험 개요

**스탠포드 감옥 실험**은 Philip Zimbardo가 진행한 사회심리학 실험이다.

평범한 참가자들을 모집한 후 동전 던지기로 죄수 역할과 교도관 역할을 나누고, 스탠포드 대학 지하에 모의 감옥을 만들어 역할극을 하게 했다.

원래는 약 2주 동안 진행될 예정이었지만, 참가자들이 역할에 지나치게 몰입하면서 비이성적이고 폭력적인 행동이 나타나 6일 만에 중단되었다.

### 13.2 실험의 의미

이 실험은 개인의 성격보다 **상황과 역할의 힘**이 행동에 강하게 작용할 수 있음을 보여주는 사례로 소개된다.

| 관련 개념  | 연결                    |
| ------ | --------------------- |
| 사회적 역할 | 교도관과 죄수 역할이 행동을 변화시킴  |
| 복종     | 권위와 규칙에 따름            |
| 동조     | 집단 분위기에 맞춰 행동함        |
| 몰개성화   | 번호, 제복, 익명성이 정체성을 약화함 |
| 책임감 분산 | 역할과 제도 뒤에 개인 책임이 숨음   |

### 13.3 한계와 논란

수업에서는 이 실험의 한계도 함께 언급되었다.

| 한계     | 설명                    |
| ------ | --------------------- |
| 윤리 문제  | 현재 기준으로는 재현하기 어려움     |
| 타당성 문제 | 실제 감옥 상황과 얼마나 유사한지 논란 |
| 신뢰성 문제 | 반복 재현이 어렵다            |
| 시대적 한계 | 1970년대 연구윤리 기준 이전의 실험 |

따라서 이 실험은 확정적 증거라기보다, **상황의 힘과 역할의 영향에 대한 논쟁을 촉발한 고전적 사례**로 이해하는 것이 적절하다.

## 14. 집단 의사결정

집단은 개인보다 더 합리적이고 좋은 결정을 내릴 것처럼 보인다. 여러 사람이 모이면 더 많은 정보와 관점을 활용할 수 있기 때문이다.

그러나 실제로는 집단이 오히려 비합리적인 결정을 내릴 수도 있다. 대표적인 현상이 **집단사고**와 **집단 극화**이다.

## 15. 집단사고

### 15.1 개념

**집단사고 groupthink**는 응집력이 강한 집단에서 나타나는 비합리적이고 비생산적인 의사결정 현상이다.

집단 구성원들이 갈등을 줄이고 의견 일치를 유지하려는 욕구가 강할 때, 비판적 사고와 대안 검토가 약화된다.

즉, 최고의 결정을 내리는 것보다 **집단의 인화단결과 만장일치**를 더 중요하게 여길 때 집단사고가 발생한다.

### 15.2 집단사고의 조건

집단사고는 다음 조건에서 잘 나타난다.

| 조건       | 설명                   |
| -------- | -------------------- |
| 강한 집단응집성 | 집단 단결과 조화를 지나치게 중시   |
| 외부와의 단절  | 외부 비판과 정보가 차단됨       |
| 다양성 부족   | 구성원들의 관점과 배경이 유사함    |
| 권위적 리더십  | 리더 의견에 반대하기 어려움      |
| 갈등 회피    | 반대 의견을 불편하게 여김       |
| 확증편향     | 기존 결정을 지지하는 정보만 받아들임 |
| 높은 스트레스  | 위기 상황에서 빠른 합의를 요구받음  |

### 15.3 집단사고의 증상

Janis는 집단사고의 증상을 크게 세 유형으로 설명한다.

| 유형       | 증상        | 설명                      |
| -------- | --------- | ----------------------- |
| 집단의 과대평가 | 불가침의 환상   | 우리 집단은 실패하지 않을 것이라고 믿음  |
| 집단의 과대평가 | 도덕성의 환상   | 우리 집단의 목적은 본래 도덕적이라고 생각 |
| 폐쇄성      | 집단 합리화    | 반대 정보가 있어도 기존 결정을 정당화   |
| 폐쇄성      | 외부집단 고정관념 | 상대 집단을 단순하고 열등하게 봄      |
| 획일성 압력   | 자기검열      | 반대 의견을 스스로 억제           |
| 획일성 압력   | 만장일치의 환상  | 침묵을 동의로 착각              |
| 획일성 압력   | 반대자 압박    | 반대 의견을 내는 사람에게 압력 행사    |

### 15.4 결함 있는 의사결정

집단사고가 발생하면 의사결정 과정에서 다음 문제가 생긴다.

| 결함            | 설명                        |
| ------------- | ------------------------- |
| 정보 수집 부족      | 결정을 내리기 전에 충분한 정보를 모으지 않음 |
| 대안 검토 부족      | 다른 행동 방안을 충분히 고려하지 않음     |
| 위험 평가 실패      | 선호하는 선택의 위험을 과소평가         |
| 거부된 대안 재검토 실패 | 처음 배제한 대안을 다시 살피지 않음      |
| 선택적 정보 처리     | 자신들의 생각에 맞는 정보만 받아들임      |
| 비상계획 부족       | 실패했을 때의 대책을 마련하지 않음       |

## 16. 피그만 침공 실패 사례

집단사고의 대표 사례로 **피그만 침공 실패**가 소개되었다.

미국 CIA는 쿠바 정권을 전복시키기 위해 쿠바인 망명자들을 훈련시킨 뒤 피그만 지역에 침투시키는 작전을 계획했다.

그러나 이 작전은 실패했다.

| 실패 요인      | 설명                         |
| ---------- | -------------------------- |
| 지형 파악 부족   | 피그만 지역의 실제 조건을 제대로 파악하지 못함 |
| 현지 지원 과대평가 | 쿠바 내부에서 봉기가 일어날 것이라고 기대    |
| 적군 과소평가    | 예상보다 강한 쿠바군에 직면            |
| 대안 부족      | 실패했을 때의 계획이 충분하지 않음        |
| 과도한 낙관     | 작전 성공을 지나치게 확신             |

Janis는 “왜 똑똑한 사람들이 모인 집단이 최악의 결정을 내릴 수 있는가?”라는 질문에서 집단사고를 설명했다.

## 17. 집단 극화

### 17.1 개념

**집단 극화 group polarization**는 개인이 혼자 결정할 때보다 집단으로 논의한 후 더 극단적인 결정을 내리는 경향이다.

집단 토론은 의견을 평균으로 완화시키는 것이 아니라, 구성원들의 초기 경향을 더 강하게 만들 수 있다.

| 초기 경향  | 집단 토론 후   |
| ------ | --------- |
| 모험적 의견 | 더 모험적인 결정 |
| 보수적 의견 | 더 보수적인 결정 |
| 강경한 의견 | 더 강경한 결정  |

### 17.2 집단사고와 집단 극화의 연결

집단사고와 집단 극화가 함께 작용하면 집단의 판단은 더 위험해질 수 있다.

```text
집단응집성 강화
→ 반대 의견 억제
→ 만장일치의 착각
→ 기존 의견 강화
→ 극단적 결정
→ 실패 가능성 증가
```

## 18. Janis의 집단사고 이론과 소련의 아프가니스탄 침공

### 18.1 논문의 핵심 주장

14-2 논문은 1979년 소련의 아프가니스탄 침공 결정을 Janis의 집단사고 이론으로 분석한다.

논문의 핵심 질문은 다음과 같다.

> 왜 소련 지도부는 아프가니스탄 침공의 국제정치적 위험성을 심각하게 과소평가했는가?

논문은 그 이유를 소련 지도부의 **집단응집성**, **일치성에 대한 압력**, **폐쇄적 의사결정**, **브레즈네프 독트린에 대한 고착**에서 찾는다. 

### 18.2 브레즈네프 독트린

**브레즈네프 독트린 Brezhnev Doctrine**은 사회주의 국가가 사회주의 체제에서 이탈하거나 반혁명 세력의 위협을 받을 경우, 소련이 군사적으로 개입할 권리와 의무가 있다는 논리이다.

핵심은 **제한된 주권 limited sovereignty**이다.

즉, 개별 국가의 주권보다 사회주의 진영 전체의 이익이 우선한다고 보는 것이다.

| 브레즈네프 독트린의 논리    | 의미                     |
| ---------------- | ---------------------- |
| 사회주의 진영의 이익 우선   | 개별 국가의 자율성은 제한됨        |
| 반사회주의 움직임은 전체 위협 | 한 국가의 변화도 소련 안보 문제로 해석 |
| 군사개입 정당화         | 체제 유지를 위해 무력 사용 가능     |
| 소련의 해석권 독점       | 무엇이 사회주의인지 소련이 판단      |

### 18.3 소련 지도부의 집단사고

소련 지도부는 아프가니스탄의 혼란을 그 나라 내부의 정치·문화·종교적 문제로 보기보다, 사회주의 진영의 이탈 가능성으로 해석했다.

이 과정에서 다음과 같은 집단사고가 나타났다.

| 집단사고 요소  | 소련 사례                          |
| -------- | ------------------------------ |
| 집단응집성    | 소비에트 체제와 사회주의 진영의 일체성을 지나치게 중시 |
| 폐쇄적 의사결정 | 브레즈네프와 소수 측근 중심으로 결정           |
| 일치성 압력   | 브레즈네프 독트린에 반하는 의견이 억제됨         |
| 위험 과소평가  | 장기전, 국제 비난, 내부 저항 가능성을 낮게 평가   |
| 대안 검토 부족 | 외교적 해결, 제한적 지원, 철수 등 대안 검토 부족  |
| 비상계획 부족  | 전쟁 장기화에 대한 현실적 계획 부족           |

### 18.4 결과

소련의 침공은 단기적으로는 아민 정권을 제거하고 친소련 카르말 정부를 세웠다. 그러나 장기적으로는 실패한 정책이었다.

| 결과         | 설명                                  |
| ---------- | ----------------------------------- |
| 국제적 비난     | UN 비난 결의, 서방의 제재, 1980 모스크바 올림픽 보이콧 |
| 미국과의 긴장 심화 | 데탕트 분위기 약화, 신냉전 분위기 강화              |
| 장기전        | 무자헤딘 저항과 산악 게릴라전으로 전쟁 장기화           |
| 소련 부담 증가   | 군사·경제적 비용 증가                        |
| 소련 체제 약화   | 이후 소련 붕괴와 냉전 종식에 영향                 |
| 아프가니스탄 불안정 | 무자헤딘, 탈레반 부상으로 이어지는 장기적 혼란          |

논문은 소련의 아프가니스탄 침공을 **집단사고에 의한 결함 있는 의사결정**으로 평가한다. 

## 19. 집단사고 방지 방법

Janis는 집단사고를 방지하기 위한 조치를 제안했다.

| 방지 방법         | 설명                      |
| ------------- | ----------------------- |
| 자유로운 의견 개진 독려 | 리더가 모든 구성원의 의견을 장려해야 함  |
| 다양성 보장        | 다른 목소리를 낼 수 있는 분위기 조성   |
| 악마의 대변인 지정    | 의도적으로 반대 의견을 제시하는 역할 부여 |
| 하위집단 토론       | 전체 집단을 나누어 독립적으로 논의     |
| 최종 결정 전 재검토   | 결정을 내리기 전에 다시 검토        |
| 외부 전문가 자문     | 집단 밖의 관점과 정보 반영         |
| 개인 책임감 강화     | 책임감이 분산되지 않도록 함         |

특히 **악마의 대변인 Devil’s Advocate**은 확증편향과 동조 압력을 줄이는 데 효과적이다.

## 20. 14주차 핵심 흐름 정리

14주차 전체 내용을 하나의 흐름으로 정리하면 다음과 같다.

```text
타인의 존재
→ 각성 증가
→ 쉬운 과제/숙련 과제: 사회적 촉진
→ 어려운 과제/미숙 과제: 사회적 억제
```

```text
집단 속 개인
→ 개인 기여도 불명확
→ 책임감 분산
→ 사회적 태만
```

```text
사람이 많은 응급상황
→ 사건 인식 어려움
→ 응급상황 해석 어려움
→ 책임감 분산
→ 도움 방법 불확실
→ 행동 비용 고려
→ 방관자 효과
```

```text
군중 속 익명성
→ 자의식 감소
→ 평가불안 감소
→ 자기조절 약화
→ 몰개성화
```

```text
응집력 강한 집단
→ 반대 의견 억제
→ 만장일치의 착각
→ 대안 검토 부족
→ 집단사고
```

## 21. 시험 대비 핵심 정리

녹음에서 안내된 시험 범위 기준으로 보면, **집단사고는 시험 범위에서 제외**된다고 언급되었다. 다만 14주차 전체 통합 정리에서는 자료에 포함되어 있으므로 함께 정리했다. 

시험 대비 우선순위는 다음과 같이 잡으면 좋다.

### 21.1 반드시 이해할 개념

| 개념            | 시험 포인트                |
| ------------- | --------------------- |
| 사회적 촉진        | 타인의 존재가 수행을 높이는 현상    |
| 사회적 억제        | 타인의 존재가 수행을 낮추는 현상    |
| 우세반응          | 타인의 존재 → 각성 → 우세반응 강화 |
| 과제 난이도        | 쉬운 과제는 촉진, 어려운 과제는 억제 |
| 사회적 태만        | 집단 속 개인 노력 감소         |
| Ringelmann 효과 | 집단 규모 증가 시 개인당 노력 감소  |
| 책임감 분산        | 사람이 많을수록 책임감 희석       |
| 방관자 효과        | 주변 사람이 많을수록 도움 행동 감소  |
| Latane 모형     | 도움 행동의 5단계            |
| 가산/결합/분리과제    | 과제 유형별 집단 수행 차이       |
| 몰개성화          | 익명성, 책임감 분산, 자의식 감소   |

### 21.2 약술형 대비 문장

#### 사회적 촉진

사회적 촉진은 타인의 존재로 개인의 수행이 향상되는 현상이다. 그러나 타인의 존재가 항상 수행을 높이는 것은 아니며, Zajonc의 우세반응 이론에 따르면 타인의 존재는 각성을 높이고 각성은 우세반응을 강화한다. 따라서 쉽거나 익숙한 과제에서는 수행이 향상되지만, 어렵거나 미숙한 과제에서는 오히려 수행이 저하될 수 있다.

#### 사회적 태만

사회적 태만은 여러 사람이 함께 과제를 수행할 때 개인의 노력이 감소하는 현상이다. 집단 과제에서는 개인의 기여도가 명확히 드러나지 않고 책임감이 분산되기 때문에 태만이 발생한다. 이를 줄이기 위해서는 개인별 기여도를 평가하고, 역할을 명확히 하며, 개인의 노력이 집단 성공에 중요하다는 인식을 부여해야 한다.

#### 방관자 효과

방관자 효과는 주변에 사람이 많을수록 도움 행동이 감소하는 현상이다. 이는 사람들이 비도덕적이어서가 아니라, 사람이 많을 때 사건 인식이 늦어지고, 상황 해석이 애매해지며, 책임감이 분산되기 때문이다. Latane와 Darley는 도움 행동이 사건 인식, 응급상황 해석, 책임감 수용, 도움 방법 결정, 실제 행동의 단계를 거쳐야 발생한다고 보았다.

#### 몰개성화

몰개성화는 집단이나 군중 속에서 개인의 정체감과 책임감이 약화되는 현상이다. 익명성, 책임감 분산, 과도한 각성, 자기구분 단서 부족이 있을 때 개인은 자의식과 평가불안이 낮아지고 자기조절이 약화된다. 그 결과 평소에는 하지 않을 충동적이거나 공격적인 행동을 할 가능성이 높아진다.

## 22. 최종 핵심 요약

14주차의 핵심은 다음 한 문장으로 정리할 수 있다.

> 개인의 행동은 개인의 성격만으로 결정되지 않으며, 타인의 존재와 집단 상황은 수행, 책임감, 도움 행동, 자기통제, 의사결정에 강력한 영향을 미친다.

마지막으로 전체 개념을 비교하면 다음과 같다.

| 개념     | 상황                  | 결과           |
| ------ | ------------------- | ------------ |
| 사회적 촉진 | 타인이 있고 과제가 쉽거나 익숙함  | 수행 향상        |
| 사회적 억제 | 타인이 있고 과제가 어렵거나 미숙함 | 수행 저하        |
| 사회적 태만 | 집단 과제에서 개인 기여가 불명확함 | 노력 감소        |
| 책임감 분산 | 여러 사람이 함께 있음        | 개인 책임감 감소    |
| 방관자 효과 | 응급상황에서 방관자가 많음      | 도움 행동 감소     |
| 몰개성화   | 익명적 군중 상황           | 자기조절 약화      |
| 사회적 역할 | 특정 역할을 맡음           | 역할에 맞는 행동 강화 |

| 집단 극화  | 비슷한 의견의 집단 토론       | 의견 극단화       |
