# 11-2. 자폐성 장애 아동 실생활적응기술 훈련을 위한 VR 기능성게임 고찰 및 행동주의 심리학적 적용 요소 제안

## 1. 논문 개요

**논문 제목**
「자폐성 장애 아동 실생활적응기술 훈련을 위한 가상현실 기능성게임 고찰 및 행동주의 심리학적 적용 요소 제안」

**저자**
백종남

**핵심 주제**
이 논문은 자폐성 장애 아동의 **실생활적응기술 훈련**을 위해 VR 기반 기능성게임을 어떻게 설계할 수 있는지 검토하고, 여기에 적용할 수 있는 **행동주의 심리학적 요소**를 제안한다. 연구는 문헌고찰을 바탕으로 VR 기능성게임의 교육적 효과, 디자인 프레임워크, 행동주의적 동기화 전략을 정리한다. 

---

# 2. 연구의 배경

## 2.1 자폐성 장애 아동의 실생활 적응 문제

자폐성 장애 아동은 DSM-5 기준으로 **사회적 의사소통 및 상호작용의 결함**, **제한적이고 반복적인 행동 패턴**을 보인다. 이로 인해 일상생활에서 사회적 상황을 이해하거나 적절히 대처하는 데 어려움을 겪는다. 

특히 자폐성 장애 아동은 다음과 같은 어려움을 가진다.

| 영역       | 어려움                       |
| -------- | ------------------------- |
| 사회적 상호작용 | 타인과의 의사소통, 협동, 규칙 따르기 어려움 |
| 상징놀이     | 일반 아동과 다른 놀이 특성, 가장놀이 어려움 |
| 정서적 개입   | 현실 상황에서 적절한 정서 반응 어려움     |
| 일반화      | 배운 기술을 새로운 상황에 적용하기 어려움   |
| 감각처리     | 현실의 복잡한 감각정보에 부담을 느낌      |

따라서 자폐성 장애 아동에게는 실제 생활과 유사하지만 안전하고 예측 가능한 환경에서 반복적으로 연습할 수 있는 훈련 매체가 필요하다.

---

## 2.2 VR이 자폐성 장애 아동에게 유용한 이유

VR은 실제와 유사한 가상환경을 제공하여 사용자가 실제 상황처럼 느끼게 하는 기술이다. 논문은 VR이 자폐성 장애 아동의 실생활적응기술 훈련에 적합한 이유를 다음과 같이 설명한다. 

| VR의 장점        | 자폐성 장애 아동에게 주는 효과        |
| ------------- | ------------------------ |
| 예측 가능한 환경 제공  | 낯설거나 갑작스러운 상황에 대한 불안 감소  |
| 감각정보 단순화      | 과도한 시각·청각 자극 부담 완화       |
| 실제와 유사한 상황 구성 | 현실 상황으로의 일반화 촉진          |
| 시각적 정보 제공     | 시각 정보처리에 강점이 있는 아동에게 효과적 |
| 반복 연습 가능      | 실생활 기술을 안전하게 반복 훈련 가능    |

즉, VR은 자폐성 장애 아동에게 **안전한 가상환경에서 현실 적응 기술을 반복적으로 익히게 하는 도구**로 활용될 수 있다.

---

# 3. VR 기능성게임의 교육적 효과

## 3.1 VR의 핵심 심리 기제: 현존감

VR의 핵심은 **현존감**이다. 현존감은 사용자가 가상환경 안에 실제로 존재하는 것처럼 느끼는 감각이다. 논문에서는 현존감이 VR 교육 콘텐츠 개발에서 반드시 고려해야 할 요인이라고 본다. 

현존감은 VR의 교육적 효과를 높이지만, 동시에 부작용도 일으킬 수 있다.

| 긍정적 측면           | 부정적 측면        |
| ---------------- | ------------- |
| 학습 몰입 증가         | 과몰입 위험        |
| 흥미와 재미 유발        | 현실감 저하 가능성    |
| 실제 상황과 유사한 훈련 가능 | 현실과 가상 구분 어려움 |
| 상호작용 경험 제공       | 일상생활 방해 가능성   |

따라서 VR 교육은 **몰입을 유도하되 과몰입을 방지하는 균형**이 필요하다.

---

## 3.2 재미와 상호작용의 중요성

논문은 VR의 특수교육 활용에서 긍정적 기대감에 영향을 미치는 주요 요인으로 **재미**와 **상호작용**을 강조한다. 페이지 4의 경로 모형에서도 재미와 상호작용이 VR의 긍정적 효과와 연결되는 요인으로 제시된다. 

자폐성 장애 아동을 위한 VR 콘텐츠는 단순히 재미만 있어서는 안 된다. 아동의 인지적·신체적 제한, 생활 맥락, 학습 목표에 맞게 구성되어야 한다.

예를 들어 다음과 같은 사회적 기술을 VR에서 훈련할 수 있다.

| 훈련 영역   | 예시                 |
| ------- | ------------------ |
| 규칙 따르기  | 줄 서기, 순서 지키기       |
| 놀이기술    | 역할놀이, 공동 놀이        |
| 협동하기    | 함께 문제 해결하기         |
| 갈등 대처   | 친구와 다툼 상황 해결하기     |
| 일상생활 기술 | 교통 이용, 물건 구매, 길 찾기 |

---

# 4. VR 기능성게임의 의미

## 4.1 기능성게임이란?

기능성게임은 단순한 오락을 넘어서 교육, 훈련, 치료, 재활 등 특정 목적을 달성하기 위해 설계된 게임이다. 논문에서는 기능성게임을 **게임의 재미 요소와 교육적 목적이 결합된 게임**으로 설명한다. 

| 구분      | 특징             |
| ------- | -------------- |
| 상업용 게임  | 재미와 오락 중심      |
| 교육용 콘텐츠 | 학습 결과 중심       |
| 기능성게임   | 재미와 목적성을 함께 추구 |

기능성게임은 학습자가 즐겁게 참여하면서도 특정 기술을 습득하도록 설계된다.

---

## 4.2 VR 기능성게임의 장점

VR 기능성게임은 VR의 현존감과 기능성게임의 교육적 목적이 결합된 형태이다. 자폐성 장애 아동에게는 다음과 같은 장점이 있다.

| 장점              | 설명                              |
| --------------- | ------------------------------- |
| 위험한 상황을 안전하게 연습 | 재난 대피, 교통 상황 등 실제로 위험한 상황 훈련 가능 |
| 반복 훈련 가능        | 같은 상황을 여러 번 연습 가능               |
| 난이도 조절 가능       | 아동 수준에 맞게 자극과 과제 난이도 조절         |
| 흥미 유발           | 게임 요소를 통해 참여 동기 강화              |
| 일반화 촉진          | 현실과 유사한 환경을 통해 실제 생활 적용 가능성 증가  |

---

# 5. VR 기능성게임 디자인

## 5.1 사용자 인터페이스 디자인

VR 기능성게임을 개발할 때는 사용자 인터페이스, 즉 UI 디자인이 중요하다. 특히 자폐성 장애 아동은 감각 자극에 민감하거나 복잡한 정보를 처리하는 데 어려움을 겪을 수 있기 때문에, UI는 단순하고 명확해야 한다.

논문은 UI 디자인 과정을 다음과 같이 제시한다. 

| 단계  | 내용                |
| --- | ----------------- |
| 1단계 | 주된 사용자 그룹 선정      |
| 2단계 | 사용자 요구 분석         |
| 3단계 | 게임 안에서 사용자의 행위 파악 |
| 4단계 | 프로토타입 작성          |
| 5단계 | 사용성 테스트           |
| 6단계 | 실제 개발             |

자폐성 장애 아동을 위한 VR 기능성게임에서는 사용자 그룹을 단순히 “초급자”로 보는 것이 아니라, 아동의 장애 특성, 인지 수준, 감각 민감성, 선호 자극 등을 구체적으로 분석해야 한다.

---

## 5.2 VR 기능성게임 개발 시 고려 질문

논문은 개발 과정에서 다음과 같은 질문에 답해야 한다고 본다.

| 질문                   | 의미                         |
| -------------------- | -------------------------- |
| 프로그램의 목적과 목표는 무엇인가?  | 어떤 실생활 기술을 훈련할 것인가         |
| 사용자의 기대를 충족시킬 수 있는가? | 아동과 교사의 요구에 맞는가            |
| 사용자의 작업환경에 맞는가?      | 실제 교육·치료 환경에서 사용할 수 있는가    |
| 사용자는 무엇을 하게 되는가?     | 게임 내 행동이 학습목표와 연결되는가       |
| 시각적 디자인은 적절한가?       | 자폐성 장애 아동에게 과도한 자극이 되지 않는가 |

# 6. VR 기능성게임 디자인 프레임워크

논문은 자폐성 장애 아동을 위한 VR 기능성게임 개발 프레임워크를 세 가지 요소로 정리한다. 페이지 10의 그림 2는 이 프레임워크를 시각적으로 보여주며, 중심 요소로 **페다고지**, **학습 내용 및 게임 메커니즘**, **평가**가 제시된다. 

## 6.1 페다고지

페다고지는 교육적 설계 원리를 의미한다. 자폐성 장애 아동을 위한 기능성게임은 단순히 게임을 만드는 것이 아니라, 명확한 학습목표와 교육적 절차를 갖추어야 한다.

페다고지의 구성 요소는 다음과 같다.

| 요소      | 설명                 |
| ------- | ------------------ |
| 교수자     | 특수교사, 치료사, 보호자 등   |
| 학습자     | 자폐성 장애 아동          |
| 학습목표    | 훈련해야 할 실생활적응기술     |
| 참여자 디자인 | 개발 과정에 전문가와 사용자 참여 |

특히 논문은 게임 개발 과정에 특수교사뿐 아니라 잠재적 사용자도 참여해야 한다고 본다. 이는 실제 사용자의 요구를 반영하기 위한 것이다.

---

## 6.2 학습 내용 및 게임 메커니즘

학습 내용은 게임을 통해 달성해야 할 목표 행동이고, 게임 메커니즘은 그 목표 행동을 재미있고 지속적으로 연습하게 만드는 구조이다.

주요 요소는 다음과 같다. 

| 요소            | 설명                            |
| ------------- | ----------------------------- |
| 몰입감           | 실제 상황처럼 느끼게 하되 과몰입은 방지        |
| 단순한 그래픽 인터페이스 | 복잡한 화면으로 인한 압도감과 주의산만 방지      |
| 개별화           | 아동의 특성과 수준에 맞는 내용 제공          |
| 자기학습          | 사용자가 선택하고 반복할 수 있는 구조         |
| 지속적 도전        | 난이도를 점진적으로 높여 동기 유지           |
| 반복            | 자폐성 장애 아동의 학습 특성에 맞춰 반복 연습 허용 |

자폐성 장애 아동을 위한 VR 기능성게임에서는 지나치게 복잡한 그래픽이나 과도한 자극을 피해야 한다. 복잡한 인터페이스는 사용자를 압도하거나 주의를 산만하게 만들어 학습을 방해할 수 있다.

---

## 6.3 평가

평가는 게임이 실제로 학습목표 달성에 도움이 되는지를 확인하는 과정이다.

논문은 평가 요소로 다음 두 가지를 제시한다.

| 평가 요소    | 설명                                   |
| -------- | ------------------------------------ |
| 사용성 평가   | 학습자와 교수자가 게임을 쉽게 사용할 수 있는지, 만족하는지 평가 |
| 게임 학습 분석 | 게임 내 행동 데이터를 관찰·측정·분석하여 학습 성과 확인     |

즉, VR 기능성게임은 단순히 재미있게 만들어지는 것에서 끝나면 안 된다. 사용자가 실제로 목표 행동을 습득했는지, 현실 생활에 적용할 수 있는지를 평가해야 한다.

# 7. VR 기능성게임 동기화를 위한 행동주의 적용

논문의 핵심 부분은 자폐성 장애 아동의 VR 기능성게임 참여를 유지하기 위해 **행동주의 원리**를 적용해야 한다는 주장이다.

행동주의는 학습자의 행동이 강화, 처벌, 강화계획, 촉진, 행동연쇄 등을 통해 형성되고 유지된다고 본다. 논문은 이를 VR 기능성게임 설계에 적용한다.

---

## 7.1 강화와 처벌의 분명한 적용

기능성게임에서 학습자가 어떤 행동을 해야 하는지 분명히 알 수 있도록 강화와 처벌이 명확하게 제시되어야 한다.

| 구분 | 게임 적용 예시                      |
| -- | ----------------------------- |
| 강화 | 올바른 행동을 하면 점수, 칭찬, 보상 아이템 제공  |
| 처벌 | 잘못된 선택을 하면 점수 감소, 재시도, 피드백 제공 |

단, 처벌은 학습자를 좌절시키기 위한 것이 아니라, 부적절한 행동을 줄이고 적절한 행동을 안내하기 위한 방식이어야 한다.

예를 들어, 횡단보도 건너기 훈련 게임에서 초록불에 건너면 칭찬과 점수를 제공하고, 빨간불에 건너려 하면 경고음과 재시도 안내를 제공할 수 있다.

---

## 7.2 강화계획 적용

강화계획은 보상을 언제, 어떤 방식으로 제공할지 정하는 체계이다. 논문은 게임 참여를 지속시키기 위해 강화계획이 필요하다고 본다. 

| 단계    | 강화 방식  |
| ----- | ------ |
| 초기 단계 | 연속 강화  |
| 이후 단계 | 간헐적 강화 |

처음에는 올바른 행동을 할 때마다 즉각적으로 보상을 제공해 게임 참여를 유도한다. 이후에는 고정비율, 변동비율, 고정간격, 변동간격 등의 강화계획을 적용하여 지속적인 참여와 도전성을 유지한다.

---

## 7.3 반복형 실습을 위한 행동연쇄 다양화

자폐성 장애 아동의 실생활 기술 학습에는 반복 연습이 필요하다. 하지만 같은 방식의 반복은 지루함을 유발할 수 있다. 그래서 논문은 **행동연쇄**를 다양화해야 한다고 제안한다.

행동연쇄는 하나의 복잡한 행동을 여러 단계로 나누어 순서대로 학습시키는 방법이다.

예를 들어, 편의점에서 물건 사기 행동은 다음과 같이 나눌 수 있다.

| 단계 | 행동         |
| -- | ---------- |
| 1  | 편의점 입장     |
| 2  | 원하는 물건 찾기  |
| 3  | 계산대로 이동    |
| 4  | 돈 또는 카드 제시 |
| 5  | 물건과 영수증 받기 |
| 6  | 인사하고 나가기   |

이를 전진법, 후진법 등 다양한 방식으로 연습하게 하면 반복 훈련이 지루하지 않고, 학습자의 흥미와 도전감을 유지할 수 있다.

---

## 7.4 자극촉진을 활용한 난이도 조절

자극촉진은 학습자가 올바른 반응을 할 수 있도록 과제 자극을 조정하는 것이다.

VR 기능성게임에서는 같은 주제라도 아동의 수준에 따라 난이도를 조절해야 한다. 예를 들어 처음에는 단순한 시각 단서와 명확한 선택지를 제공하고, 점차 현실과 유사하게 복잡도를 높일 수 있다.

| 난이도    | 예시                     |
| ------ | ---------------------- |
| 쉬운 단계  | 정답 선택지가 밝게 표시됨         |
| 중간 단계  | 힌트가 줄어듦                |
| 어려운 단계 | 실제 상황처럼 여러 자극 중 스스로 판단 |
| 일반화 단계 | 장소, 사람, 상황을 바꾸어 적용     |

이 방식은 자폐성 장애 아동이 처음에는 쉽게 성공 경험을 쌓고, 이후 더 복잡한 현실 상황으로 자연스럽게 나아가도록 돕는다.

---

## 7.5 개인 선호에 따른 반응촉진 다양화

반응촉진은 학습자가 올바른 반응을 하기 전에 도움을 제공하는 것이다. 자폐성 장애 아동은 개인마다 선호하는 촉진 방식이 다르므로, 다양한 방식의 촉진을 제공해야 한다.

| 촉진 방식  | 예시                 |
| ------ | ------------------ |
| 시각 촉진  | 그림, 아이콘, 화살표, 색 표시 |
| 청각 촉진  | 음성 안내, 효과음         |
| 비디오 촉진 | 짧은 시범 영상           |
| 몸짓 촉진  | 캐릭터의 손짓, 방향 안내     |
| 텍스트 촉진 | 간단한 문장 안내          |

중요한 점은 촉진을 계속 제공하는 것이 아니라, 점차 줄여 나가야 한다는 것이다. 이를 **촉진의 용암**이라고 한다. 최종 목표는 아동이 도움 없이 스스로 과제를 수행하는 것이다.

# 9. 논문의 의의

이 논문의 의의는 VR 기능성게임을 단순한 기술 도구로 보지 않고, **특수교육**, **게임 디자인**, **행동주의 심리학**을 결합해 자폐성 장애 아동의 실생활 기술 훈련 방안을 제안했다는 점이다.

특히 자폐성 장애 아동의 특성에 맞춰 다음을 강조한 점이 중요하다.

첫째, 실제 상황과 유사하지만 안전하고 예측 가능한 VR 환경을 활용한다.
둘째, 게임의 재미와 몰입을 학습 동기로 활용한다.
셋째, 반복 연습과 일반화를 위해 행동주의 원리를 적용한다.
넷째, 강화와 촉진을 통해 목표 행동을 체계적으로 형성한다.
다섯째, 사용성 평가와 학습 분석을 통해 효과를 검증해야 한다.

# 12. 최종 요약

이 논문은 자폐성 장애 아동의 실생활적응기술 훈련을 위해 VR 기능성게임을 활용할 수 있다고 본다. 자폐성 장애 아동은 사회적 상호작용, 일반화, 감각정보 처리에서 어려움을 겪기 때문에, 안전하고 예측 가능하며 반복 연습이 가능한 VR 환경이 효과적인 훈련 도구가 될 수 있다.

VR 기능성게임은 재미, 몰입, 상호작용을 통해 아동의 참여를 유도할 수 있다. 하지만 과몰입의 위험도 있기 때문에, 교육적 목표에 맞게 적절히 설계되어야 한다.

논문은 VR 기능성게임 개발 프레임워크로 **페다고지**, **학습 내용 및 게임 메커니즘**, **평가**를 제시한다. 또한 자폐성 장애 아동의 학습 동기를 유지하기 위해 **강화와 처벌의 분명한 적용**, **강화계획**, **행동연쇄 다양화**, **자극촉진**, **반응촉진**이라는 행동주의 요소를 적용해야 한다고 제안한다.

결국 이 논문의 핵심은 VR 기능성게임이 자폐성 장애 아동의 자립생활훈련에 유용할 수 있지만, 단순한 기술 활용이 아니라 **특수교육적 목표**, **사용자 중심 디자인**, **행동주의적 학습 원리**, **체계적 평가**가 함께 반영되어야 한다는 것이다.
