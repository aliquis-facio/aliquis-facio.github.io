# 성격의 인지적 관점과 행동주의적 관점

## 1. 10주차 핵심 주제

10주차 내용은 크게 두 축으로 정리할 수 있다.

첫째는 **성격의 인지적 관점**이다.
인지적 관점은 인간의 성격과 행동을 이해할 때, 외부 사건 자체보다 그 사건을 개인이 어떻게 **해석**, **평가**, **의미 부여**하는지를 중요하게 본다.

둘째는 **성격의 행동주의적 관점**이다.
행동주의적 관점은 성격을 내면의 고정된 특성으로 보기보다, 환경 속에서 반복적으로 학습된 **행동 패턴**으로 본다.

즉, 10주차의 핵심은 다음과 같다.

```text
인지적 관점:
사건 자체보다 사건에 대한 해석과 신념이 정서와 행동을 결정한다.

행동주의적 관점:
성격은 환경 속에서 강화와 처벌을 통해 학습된 행동 패턴이다.
```

# 2. 성격의 인지적 관점

## 2.1 인지적 관점의 기본 입장

인지적 관점은 인간이 현실을 있는 그대로 반응하는 존재가 아니라, 현실을 마음속에서 해석하고 구성하는 존재라고 본다.

같은 사건을 경험하더라도 사람마다 다르게 느끼고 행동하는 이유는 각자가 가진 다음 요소가 다르기 때문이다.

| 요소    | 의미                           |
| ----- | ---------------------------- |
| 신념    | 사건과 자신, 타인, 세상에 대해 가지고 있는 믿음 |
| 기대    | 앞으로 어떤 일이 일어날 것이라고 예상하는 방식   |
| 해석 방식 | 사건에 의미를 부여하는 방식              |
| 사고 과정 | 정보를 처리하고 판단하는 방식             |
| 인지 도식 | 자기와 세상을 이해하는 기본 틀            |

따라서 성격을 이해하려면 단순히 겉으로 드러난 행동만 볼 것이 아니라, 그 사람이 세상을 어떻게 해석하는지를 살펴보아야 한다.

## 2.2 인지, 정서, 행동의 관계

인지적 관점에서는 **인지가 정서와 행동에 영향을 준다**고 본다.

```text
사건 → 해석/신념 → 감정 → 행동
```

예를 들어 친구가 답장을 하지 않았을 때, 해석에 따라 감정과 행동이 달라진다.

```text
상황:
친구가 메시지에 답장하지 않음

해석 1:
“나를 무시하는 게 분명해.”
→ 분노, 불안, 관계 회피

해석 2:
“바빠서 아직 못 봤을 수도 있어.”
→ 기다림, 비교적 안정된 감정
```

즉, 인간의 감정과 행동은 사건 자체보다 사건에 대한 **인지적 해석**에 의해 크게 달라진다.

# 3. Ellis의 ABC 모델

## 3.1 ABC 모델의 구조

Ellis는 인간의 정서와 행동을 설명하기 위해 **ABC 모델**을 제시했다.

| 단계 | 의미               | 설명                |
| -- | ---------------- | ----------------- |
| A  | Activating Event | 촉발 사건, 선행 사건      |
| B  | Belief           | 사건에 대한 신념, 해석, 평가 |
| C  | Consequence      | 정서적·행동적 결과        |

ABC 모델의 핵심은 **A가 직접 C를 만드는 것이 아니라, B가 A와 C 사이를 매개한다**는 것이다.

```text
A: 사건
B: 사건에 대한 신념
C: 감정과 행동 결과
```

즉, 사람들은 사건 자체 때문에 고통받는 것이 아니라, 사건에 대한 자신의 생각과 신념 때문에 고통받는다.

## 3.2 ABC 모델 예시

```text
A: 발표 중 실수함
B: “나는 절대 실수하면 안 된다. 사람들이 나를 무능하다고 볼 것이다.”
C: 불안, 수치심, 발표 회피
```

같은 사건이라도 B가 바뀌면 결과가 달라진다.

```text
A: 발표 중 실수함
B: “실수할 수 있다. 그래도 발표 전체가 망한 것은 아니다.”
C: 긴장되지만 발표를 계속함
```

이처럼 REBT에서는 정서적 문제를 해결하기 위해 사건을 바꾸기보다 **신념을 점검하고 수정하는 것**을 중요하게 본다.

# 4. 합리적 사고와 비합리적 사고

## 4.1 Ellis가 본 인간

Ellis는 인간을 **합리적이면서 동시에 비합리적인 존재**로 보았다.

인간은 합리적으로 사고할 수 있는 능력을 가지고 있지만, 동시에 스스로를 괴롭히는 방식으로 비합리적 사고를 하기도 한다.

특히 인간은 자신이 원하는 것을 얻지 못할 때 다음 대상을 비난하는 경향이 있다.

* 자신
* 타인
* 세상

이러한 사고는 자기 패배적 사고로 이어질 수 있다.

## 4.2 합리적 사고의 기준

Ellis가 말한 합리적 사고는 단순히 도덕적으로 옳거나 완벽한 생각을 의미하지 않는다.

중요한 기준은 다음과 같다.

| 기준    | 질문                     |
| ----- | ---------------------- |
| 현실성   | 이 생각은 현실과 맞는가?         |
| 융통성   | 이 생각은 유연한가?            |
| 실용성   | 이 생각은 문제 해결에 도움이 되는가?  |
| 정신건강  | 이 생각은 정서적 안정에 도움이 되는가? |
| 목표 달성 | 이 생각은 삶의 목표에 도움이 되는가?  |

핵심은 다음 질문이다.

```text
“이 생각이 옳은가?”보다
“이 생각이 나에게 도움이 되는가?”
```

# 5. 비합리적 신념

## 5.1 비합리적 신념의 의미

비합리적 신념은 현실에서 충족되기 어려운 기대와 요구를 의미한다.

대개 다음과 같은 표현으로 나타난다.

* 반드시 ~해야 한다
* 꼭 그래야 한다
* 항상 그래야 한다
* 절대로 ~하면 안 된다
* 예외 없이 그래야 한다

이러한 사고는 당위적이고 경직된 완벽주의적 사고이다.

## 5.2 비합리적 신념의 세 가지 당위성

Ellis는 비합리적 신념이 주로 세 가지 방향으로 나타난다고 보았다.

| 구분            | 설명                            | 예시                       |
| ------------- | ----------------------------- | ------------------------ |
| 자신에 대한 당위성    | 내가 반드시 완벽해야 한다는 요구            | “나는 항상 인정받아야 한다.”        |
| 타인에 대한 당위성    | 타인이 반드시 나에게 잘해야 한다는 요구        | “진정한 친구라면 항상 내 편이어야 한다.” |
| 조건/세상에 대한 당위성 | 세상이 반드시 공정하고 내 뜻대로 되어야 한다는 요구 | “세상은 항상 정의로워야 한다.”       |

## 5.3 핵심 비합리적 신념의 형태

비합리적 신념은 다음과 같은 방식으로 나타난다.

| 유형        | 설명                 | 예시                            |
| --------- | ------------------ | ----------------------------- |
| 절대적 당위    | 반드시 그래야 한다는 사고     | “나는 절대 실패하면 안 된다.”            |
| 파국화       | 사건을 끔찍하게 과장함       | “이건 인생이 끝난 것이다.”              |
| 낮은 좌절 인내력 | 불편함을 견딜 수 없다고 생각함  | “이런 일은 도저히 못 참겠다.”            |
| 자기·타인 질책  | 행동이 아니라 인격 전체를 비난함 | “나는 무가치하다.”, “그 사람은 나쁜 인간이다.” |

# 6. REBT와 ABCDEF 모델

## 6.1 REBT의 핵심

REBT, 즉 합리적 정서행동치료는 비합리적 신념을 찾아내고 논박하여 더 합리적인 신념으로 바꾸는 상담 접근이다.

REBT의 목적은 단순히 기분을 좋게 만드는 것이 아니라, 개인이 더 현실적이고 유연하며 실용적인 사고를 하도록 돕는 것이다.

## 6.2 ABCDEF 모델

ABC 모델은 치료 과정에서 ABCDEF 모델로 확장된다.

| 단계 | 의미                   | 설명         |
| -- | -------------------- | ---------- |
| A  | Activating Event     | 촉발 사건      |
| B  | Belief               | 비합리적 신념    |
| C  | Consequence          | 정서적·행동적 결과 |
| D  | Dispute              | 신념 논박      |
| E  | Effect               | 효과적인 새 인지  |
| F  | New Feeling/Behavior | 새로운 감정과 행동 |

예시는 다음과 같다.

```text
A: 친구가 약속을 취소함
B: “나를 중요하게 생각하지 않는 거야. 진짜 친구라면 이러면 안 돼.”
C: 분노, 서운함, 관계 단절
D: 정말 친구가 나를 무시했다는 증거가 있는가?
E: 사정이 있었을 수도 있다. 서운하지만 관계 전체를 단정할 필요는 없다.
F: 감정을 조절하고 대화로 확인함
```

# 7. REBT의 논박 기준

비합리적 신념을 논박할 때는 다음 기준을 사용할 수 있다.

## 7.1 현실성 논박

```text
정말 항상 완벽해야만 하는가?
모든 사람이 나를 좋아하는 것이 가능한가?
실수했다고 해서 내가 무가치한 사람인가?
```

## 7.2 실용성 논박

```text
이 생각이 내 문제 해결에 도움이 되는가?
이 생각이 나를 더 건강하게 만드는가?
오히려 불안과 회피를 키우는 것은 아닌가?
```

## 7.3 융통성 논박

```text
다른 해석은 불가능한가?
반드시 그래야 하는가?
상황에 따라 조정할 수는 없는가?
```

## 7.4 자기수용적 논박

```text
내가 실수했다는 사실과 내가 가치 없다는 것은 같은 말인가?
나는 부족한 점이 있어도 여전히 존중받을 수 있는 사람인가?
```

# 8. Beck의 인지적 오류

Beck은 심리적 문제가 역기능적 인지 도식과 인지적 왜곡에서 비롯될 수 있다고 보았다.

## 8.1 역기능적 인지 도식

인지 도식은 자기와 세상을 이해하는 틀이다.

하지만 이 도식이 부정적이면 상황을 왜곡해서 해석하게 되고, 심리적 문제의 원인이 될 수 있다.

## 8.2 주요 인지적 왜곡

| 인지적 왜곡  | 설명                    | 예시                           |
| ------- | --------------------- | ---------------------------- |
| 이분법적 사고 | 성공 아니면 실패처럼 극단적으로 판단  | “완벽하지 않으면 실패다.”              |
| 선택적 추론  | 부정적 요소 하나만 보고 전체를 판단  | “발표 중 한 번 실수했으니 완전히 망했다.”    |
| 파국화     | 최악의 결과를 과장해서 예상       | “시험을 못 보면 인생이 끝이다.”          |
| 과잉일반화   | 하나의 사건을 전체로 확대        | “한 번 실패했으니 나는 항상 실패한다.”      |
| 개인화     | 근거 없이 모든 일을 자기 탓으로 돌림 | “친구가 기분 나쁜 건 나 때문일 거야.”      |
| 정서적 추론  | 감정을 사실로 믿음            | “내가 무가치하게 느껴지니까 실제로도 무가치하다.” |

# 9. 사회불안과 인지적 관점

## 9.1 사회불안의 의미

사회불안은 사회적 상황에서 과도한 두려움을 느끼고, 타인의 부정적 평가를 지나치게 걱정하는 심리적 상태이다.

사회불안을 경험하는 사람은 다음과 같은 특징을 보인다.

* 타인이 자신을 부정적으로 평가할 것이라고 걱정함
* 자신의 말이나 행동이 실수로 보일까 두려워함
* 사회적 상황을 회피함
* 모호한 피드백도 부정적으로 해석함
* 긍정적 피드백보다 부정적 피드백을 더 잘 기억함
* 자신의 사회적 능력을 과소평가함

## 9.2 사회불안의 악순환

사회불안은 다음과 같은 악순환으로 유지된다.

```text
부정적 자동적 사고
→ 불안
→ 회피 행동
→ 사회적 경험 부족
→ 자기효능감 저하
→ 더 큰 사회불안
```

예를 들어 발표 상황에서 다음과 같은 과정이 나타날 수 있다.

```text
A: 발표 중 말을 더듬음
B: “사람들이 나를 이상하게 볼 거야.”
C: 불안, 수치심, 발표 회피
```

## 9.3 REBT의 사회불안 치료

REBT는 사회불안의 핵심 원인을 비합리적 신념에서 찾는다.

사회불안과 관련된 대표적 비합리적 신념은 다음과 같다.

```text
나는 절대 실수하면 안 된다.
모든 사람에게 좋은 평가를 받아야 한다.
누군가 나를 이상하게 보면 견딜 수 없다.
발표를 못하면 나는 무능한 사람이다.
```

REBT는 이러한 신념을 논박하여 더 합리적인 신념으로 전환하게 돕는다.

```text
실수는 불편하지만 참을 수 없는 일은 아니다.
모든 사람에게 인정받을 필요는 없다.
한 번의 발표가 내 전체 가치를 결정하지 않는다.
```

# 10. 심리극과 사회불안

## 10.1 심리극의 의미

심리극은 Moreno가 창안한 집단 심리치료 방법으로, 말로만 문제를 다루는 것이 아니라 내담자의 경험을 **무대 위에서 행위로 재현**한다.

심리극은 인지, 정서, 행동을 실제 경험 속에서 다룬다는 점에서 사회불안 치료에 활용 가능성이 있다.

## 10.2 심리극의 구성요소

| 구성요소  | 설명                      |
| ----- | ----------------------- |
| 주인공   | 자신의 문제를 표현하는 중심 인물      |
| 디렉터   | 치료 과정을 이끄는 진행자          |
| 보조 자아 | 주인공의 중요한 타인이나 내면 일부를 연기 |
| 관객    | 관찰하고 피드백을 제공하는 집단원      |
| 무대    | 내면세계와 현실을 연결하는 공간       |

## 10.3 심리극의 단계

| 단계  | 설명                       |
| --- | ------------------------ |
| 준비  | 집단 분위기 형성, 불안 감소, 주인공 선정 |
| 행위화 | 갈등 상황을 무대 위에서 재현         |
| 나누기 | 통찰과 감정을 공유하고 일상 적용을 계획   |

## 10.4 심리극의 주요 기법

| 기법    | 설명                     |
| ----- | ---------------------- |
| 역할 교대 | 상대방의 입장이 되어 상황을 바라봄    |
| 이중 자아 | 표현하지 못한 감정과 생각을 대신 드러냄 |
| 거울 기법 | 자신의 모습을 밖에서 관찰하게 함     |

심리극은 사회불안을 가진 사람이 안전한 환경에서 두려운 상황을 재현하고, 새로운 행동을 시도하며, 사회적 자기효능감을 높이도록 돕는다.

# 11. REBT와 심리극의 절충적 접근

REBT와 심리극은 서로 다른 강점을 가진다.

## 11.1 REBT의 강점과 한계

| 강점                 | 한계                                 |
| ------------------ | ---------------------------------- |
| 비합리적 신념을 논리적으로 다룸  | 정서적 경험을 충분히 다루기 어려울 수 있음           |
| ABCDE 모델로 구조화 가능   | 실제 행동 실천으로 이어지는 데 한계가 있을 수 있음      |
| 자기수용과 불편감 감내 훈련 가능 | 외상 경험이 있는 내담자에게는 인지 논박만으로 부족할 수 있음 |

## 11.2 심리극의 강점과 한계

| 강점                     | 한계                    |
| ---------------------- | --------------------- |
| 정서 표현과 정서 안정화에 유용      | 구조화가 부족하면 재외상화 위험이 있음 |
| 실제와 유사한 상황에서 행동 연습 가능  | 경험 후 인지적 정리가 부족할 수 있음 |
| 집단 피드백을 통해 자기효능감 향상 가능 | 상담자의 높은 전문성이 요구됨      |

## 11.3 절충적 접근의 의의

REBT와 심리극을 결합하면 다음과 같은 장점이 있다.

```text
REBT:
비합리적 신념을 찾고 논박한다.

심리극:
그 신념과 감정을 실제 장면에서 경험하고 행동으로 연습한다.
```

즉, 두 접근은 다음과 같이 상호보완된다.

| REBT     | 심리극                 |
| -------- | ------------------- |
| 인지적 재구성  | 정서적 안정화             |
| 논리적 논박   | 행위와 경험              |
| 자기수용     | 사회적 자기효능감           |
| ABCDE 구조 | 역할 교대, 이중 자아, 거울 기법 |

# 12. Ellis 비합리적 신념 측정 논문의 핵심

## 12.1 연구의 문제의식

Ellis의 비합리적 신념은 REBT의 핵심 개념이지만, 기존 측정도구들은 이 개념을 충분히 정확하게 측정하지 못한다는 문제가 있다.

기존 측정도구의 문제는 다음과 같다.

* 비합리적 신념의 하위요인이 명확하지 않음
* 사고 내용과 정서·행동 결과가 혼동됨
* 특정 상황에 치우친 문항이 많음
* 문항 간 개념 중복이 많음
* 자기보고식 응답의 한계가 큼
* 실제 자동적 사고 과정을 충분히 드러내지 못함

## 12.2 비합리적 신념 측정의 핵심 기준

비합리적 신념을 평가할 때는 다음 기준이 필요하다.

| 기준  | 설명                    |
| --- | --------------------- |
| 논리성 | 사고가 논리적으로 타당한가        |
| 현실성 | 경험적 현실과 일치하는가         |
| 융통성 | 경직되어 있지 않고 유연한가       |
| 실용성 | 삶의 목표와 문제 해결에 도움이 되는가 |

## 12.3 대안적 측정틀

논문은 기존의 단순 자기보고식 문항보다 **자기반영적 측정 방식**이 필요하다고 제안한다.

```text
1. 구체적 상황 제시
2. 그 상황에서 떠오르는 자동적 사고 확인
3. 사고 속 비합리적 신념 분석
4. 논리성, 현실성, 융통성, 실용성 평가
```

예시는 다음과 같다.

```text
상황:
발표 중 말을 더듬었다.

자동적 사고:
“나는 완전히 실패했다. 사람들이 나를 무능하게 볼 것이다.”

평가:
논리적인가?
현실적인가?
융통성이 있는가?
내 목표에 도움이 되는가?
```

이 방식은 실제 상담이나 자기이해 과정에서 비합리적 신념을 더 구체적으로 파악하는 데 유용하다.

# 13. 성격의 행동주의적 관점

## 13.1 행동주의의 기본 입장

행동주의는 성격을 내면의 구조나 무의식으로 설명하기보다, **관찰 가능한 행동**으로 설명한다.

행동주의에서 인간의 마음속 과정은 직접 관찰하기 어렵기 때문에 일종의 **블랙박스**로 간주된다.

```text
자극 → [마음/생각: 블랙박스] → 반응
```

행동주의는 이 중에서 관찰 가능한 자극과 반응, 행동과 결과에 초점을 둔다.

## 13.2 행동주의에서 성격이란?

행동주의에서 성격은 다음과 같다.

```text
성격 = 환경 속에서 학습된 일관된 행동 패턴
```

즉, 외향성, 내향성, 회피성, 공격성, 친절함 등도 타고난 본질이라기보다, 과거에 어떤 행동이 강화되었는지에 따라 형성된 패턴으로 이해한다.

# 14. 행동주의에서 학습의 의미

심리학에서 학습은 공부만을 의미하지 않는다.

학습은 경험을 통해 행동이 변화하는 것이다.

```text
학습 = 경험을 통해 획득되고 변화된 행동
```

따라서 성격도 반복된 경험을 통해 학습된 결과라고 본다.

예를 들어 누군가 사람들 앞에서 말했을 때 칭찬과 관심을 받았다면, 사회적 행동이 강화될 수 있다.

반대로 말했을 때 비난이나 민망함을 경험했다면, 말하는 행동이 줄어들 수 있다.

# 15. 고전적 조건화

## 15.1 고전적 조건화의 의미

고전적 조건화는 **자극과 자극의 연합**을 통해 반응이 학습되는 과정이다.

대표적인 예시는 파블로프의 개 실험이다.

## 15.2 파블로프의 개 실험

| 단계   | 자극       | 반응    |
| ---- | -------- | ----- |
| 학습 전 | 음식       | 침 분비  |
| 학습 전 | 종소리      | 반응 없음 |
| 학습 중 | 종소리 + 음식 | 침 분비  |
| 학습 후 | 종소리      | 침 분비  |

처음에는 종소리가 아무 의미 없는 중성 자극이었지만, 음식과 반복적으로 연결되면서 침을 흘리게 하는 조건 자극이 되었다.

## 15.3 주요 개념

| 개념         | 의미                  | 예시        |
| ---------- | ------------------- | --------- |
| 무조건 자극, US | 원래 반응을 일으키는 자극      | 음식        |
| 무조건 반응, UR | 자연스럽게 나타나는 반응       | 침 분비      |
| 중성 자극, NS  | 처음에는 반응을 일으키지 않는 자극 | 종소리       |
| 조건 자극, CS  | 학습 후 반응을 일으키는 자극    | 학습 후 종소리  |
| 조건 반응, CR  | 조건 자극에 대한 학습된 반응    | 종소리에 침 흘림 |

# 16. 공포 조건화와 자극 일반화

## 16.1 공포 조건화

공포 조건화는 중립적 자극이 공포 경험과 연결되어 공포 반응을 유발하게 되는 과정이다.

예를 들어 병원에서 아픈 주사를 맞은 경험이 반복되면 병원 자체가 공포 자극이 될 수 있다.

```text
병원 + 주사 통증 → 병원에 대한 공포
```

## 16.2 Little Albert 실험

Little Albert 실험은 공포가 학습될 수 있음을 보여준다.

```text
흰쥐 + 큰 소리 → 공포
흰쥐만 제시 → 공포
```

Albert는 원래 흰쥐를 무서워하지 않았지만, 흰쥐와 큰 소리가 반복적으로 연결되자 흰쥐만 봐도 공포 반응을 보였다.

## 16.3 자극 일반화

자극 일반화는 특정 자극에 대한 반응이 유사한 자극으로 확장되는 것이다.

```text
흰쥐 공포 → 털이 있는 대상 전체에 대한 공포
```

강의에서는 이를 다음 속담과 연결했다.

```text
자라 보고 놀란 가슴 솥뚜껑 보고 놀란다
```

# 17. 역조건화

역조건화는 이미 형성된 부정적 조건 반응을 새로운 긍정적 자극과 연결하여 바꾸는 과정이다.

예를 들어 청소기를 무서워하는 강아지에게 청소기를 조금씩 보여주면서 간식을 제공하면, 청소기에 대한 공포가 줄어들 수 있다.

```text
청소기 + 간식 → 청소기에 대한 긍정적 반응
```

핵심은 공포 대상과 긍정적 경험을 점진적으로 연결하여 새로운 반응을 학습시키는 것이다.

# 18. 조작적 조건화

## 18.1 조작적 조건화의 의미

조작적 조건화는 행동의 결과에 따라 그 행동이 증가하거나 감소하는 과정이다.

```text
행동 → 결과 → 행동 증가 또는 감소
```

고전적 조건화가 자극과 자극의 연합이라면, 조작적 조건화는 **행동과 결과의 연결**이다.

## 18.2 Skinner 상자

Skinner는 쥐가 지렛대를 누르면 먹이가 나오도록 했다.

처음에는 우연히 지렛대를 눌렀지만, 먹이가 나온다는 결과를 경험한 뒤에는 지렛대를 더 자주 누르게 되었다.

```text
지렛대 누르기 → 먹이 제공 → 지렛대 누르기 증가
```

# 19. 강화와 처벌

## 19.1 기본 개념

| 개념 | 의미       |
| -- | -------- |
| 강화 | 행동을 증가시킴 |
| 처벌 | 행동을 감소시킴 |
| 정적 | 무언가를 제공함 |
| 부적 | 무언가를 제거함 |

## 19.2 네 가지 유형

| 구분    | 설명               | 예시                |
| ----- | ---------------- | ----------------- |
| 정적 강화 | 좋은 것을 제공하여 행동 증가 | 방 청소를 했더니 용돈을 받음  |
| 부적 강화 | 싫은 것을 제거하여 행동 증가 | 안전벨트를 매니 경고음이 사라짐 |
| 정적 처벌 | 싫은 것을 제공하여 행동 감소 | 주차 위반 후 벌금 부과     |
| 부적 처벌 | 좋은 것을 제거하여 행동 감소 | 규칙 위반 후 게임 금지     |

## 19.3 부적 강화와 처벌의 차이

부적 강화는 행동을 증가시킨다.

```text
두통약을 먹음 → 두통이 사라짐 → 다음에도 두통약을 먹음
```

처벌은 행동을 감소시킨다.

```text
규칙 위반 → 게임 금지 → 규칙 위반 감소
```

# 20. 행동 조성

행동 조성은 목표 행동이 나타나도록 점진적으로 유도하는 과정이다.

처음부터 목표 행동을 완벽하게 요구하지 않고, 목표에 가까운 행동을 할 때마다 강화한다.

예를 들어 공부 시간을 늘릴 때 다음과 같이 할 수 있다.

```text
5분 공부 → 보상
12분 공부 → 보상
20분 공부 → 보상
1시간 공부 → 보상
```

이 방식은 교육, 훈육, 소비자 심리, 앱 서비스 설계, 게임 설계 등에서 활용된다.

# 21. 강화 계획

## 21.1 강화 계획의 의미

강화 계획은 보상을 언제, 어떤 방식으로 제공할지를 정하는 것이다.

같은 보상이라도 제공 방식에 따라 행동의 학습 속도와 유지력이 달라진다.

## 21.2 연속 강화와 간헐적 강화

| 구분     | 설명            | 특징                 |
| ------ | ------------- | ------------------ |
| 연속 강화  | 행동할 때마다 보상 제공 | 학습은 빠르지만 소거도 빠름    |
| 간헐적 강화 | 가끔 보상 제공      | 행동 유지에 강하고 소거에 저항적 |

연속 강화는 처음 행동을 학습시킬 때 효과적이다.

그러나 행동을 오래 유지하려면 간헐적 강화가 더 효과적이다.

# 22. 간헐적 강화의 네 가지 유형

| 유형    | 기준 | 설명               | 예시                    |
| ----- | -- | ---------------- | --------------------- |
| 고정 간격 | 시간 | 정해진 시간이 지나면 보상   | 중간고사, 기말고사, 매주 금요일 할인 |
| 변동 간격 | 시간 | 언제 보상이 올지 모름     | 깜짝 퀴즈, 불규칙한 세일        |
| 고정 비율 | 횟수 | 정해진 횟수 후 보상      | 커피 10잔 후 1잔 무료        |
| 변동 비율 | 횟수 | 몇 번 해야 보상이 올지 모름 | 슬롯머신, 랜덤박스, SNS 쇼츠    |

## 22.1 변동 비율의 중요성

변동 비율은 행동을 가장 강하게 유지시키는 방식이다.

보상이 언제 나올지 모르기 때문에 행동이 쉽게 멈추지 않는다.

예시는 다음과 같다.

* 슬롯머신
* 랜덤 뽑기
* 포켓몬 빵 띠부씰
* 킨더조이 장난감
* SNS 쇼츠 스크롤
* 게임 아이템 뽑기

이 원리는 중독 행동을 설명할 때 중요하다.

# 23. 행동주의와 스마트폰/SNS 중독

강의에서는 스마트폰과 SNS 사용도 행동주의적으로 설명할 수 있다고 보았다.

SNS를 스크롤할 때마다 항상 재미있는 콘텐츠가 나오는 것은 아니다.

하지만 가끔 흥미로운 콘텐츠가 나타난다.

```text
스크롤 → 별로인 콘텐츠
스크롤 → 별로인 콘텐츠
스크롤 → 재미있는 콘텐츠
```

이처럼 보상이 불규칙하게 제공되면 행동이 지속된다.

이는 변동 비율 강화와 유사하다.

도박에서 슬롯머신을 계속 당기는 행동도 같은 원리로 설명할 수 있다.

# 24. 행동주의와 훈육

행동주의에서 훈육의 핵심은 **말보다 결과**이다.

단순히 “하지 마”라고 말하는 것만으로는 행동이 잘 변하지 않는다.

행동 뒤에 실제 결과가 따라와야 한다.

예를 들어 아이가 마트에서 떼를 쓸 때 부모가 가끔 장난감을 사주면, 떼쓰기 행동은 강화된다.

특히 어떤 때는 사주고 어떤 때는 안 사주면 변동 비율 강화가 되어 떼쓰기 행동이 더 오래 지속될 수 있다.

```text
떼쓰기 → 가끔 장난감 획득 → 떼쓰기 행동 유지
```

따라서 훈육에서는 일관성이 중요하다.

# 26. 두 관점의 연결

인지적 관점과 행동주의적 관점은 서로 달라 보이지만, 인간의 성격과 행동을 설명하는 데 함께 활용될 수 있다.

예를 들어 사회불안을 보면 다음과 같다.

## 26.1 인지적 설명

```text
발표 상황 → “나는 절대 실수하면 안 돼” → 불안 → 회피
```

인지적 관점에서는 비합리적 신념이 불안을 만든다고 본다.

## 26.2 행동주의적 설명

```text
발표 회피 → 불안 감소 → 회피 행동 강화
```

행동주의적 관점에서는 발표를 피했더니 불안이 줄어든 경험이 회피 행동을 강화한다고 본다.

즉, 사회불안은 다음처럼 통합적으로 이해할 수 있다.

```text
비합리적 신념이 불안을 만들고,
회피 행동은 불안을 일시적으로 줄여서 강화된다.
```

따라서 치료에서는 인지적 신념을 바꾸는 것과 회피 행동을 줄이는 것이 모두 중요하다.

# 28. 최종 요약

10주차에서는 성격을 이해하는 두 가지 중요한 관점을 다루었다.

**인지적 관점**은 인간의 정서와 행동이 사건 자체보다 그 사건을 어떻게 해석하는지에 의해 결정된다고 본다. Ellis의 ABC 모델은 사건과 결과 사이에 신념이 있음을 설명하며, 비합리적 신념은 불안, 우울, 분노, 회피 행동을 유발할 수 있다. REBT는 이러한 비합리적 신념을 논박하고 합리적 신념으로 바꾸는 치료적 접근이다.

**행동주의적 관점**은 성격을 환경 속에서 학습된 행동 패턴으로 본다. 고전적 조건화는 자극과 자극의 연합을 통해 공포나 정서 반응이 학습되는 과정을 설명하고, 조작적 조건화는 행동 뒤에 오는 보상과 처벌이 행동의 빈도를 변화시킨다고 본다. 강화 계획, 특히 변동 비율 강화는 도박, SNS, 랜덤 뽑기처럼 행동을 오래 유지시키는 원리로 설명된다.

결국 10주차의 핵심은 다음과 같이 정리할 수 있다.

```text
성격은 한편으로는 개인이 세상을 해석하는 인지적 신념 체계이고,
다른 한편으로는 환경 속에서 반복적으로 강화된 행동 패턴이다.
```

따라서 성격을 이해하고 변화시키기 위해서는 **내가 어떤 신념으로 사건을 해석하는지**와 **어떤 행동이 어떤 결과에 의해 강화되어 왔는지**를 함께 살펴보아야 한다.
