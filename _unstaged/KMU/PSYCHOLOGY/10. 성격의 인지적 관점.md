# 성격의 인지적 관점과 REBT

## 1. 10주차 핵심 주제

10주차의 중심 주제는 **성격의 인지적 관점**이다.

인지적 관점은 인간의 성격과 행동을 이해할 때, 사건 자체보다 그 사건을 개인이 어떻게 **해석**, **판단**, **의미 부여**하는지를 중요하게 본다.

```text
사건 → 해석 / 신념 → 감정과 행동
```

즉, 같은 사건을 경험하더라도 사람마다 다르게 느끼고 행동하는 이유는 각자가 가진 **인지적 해석 방식**이 다르기 때문이다.

## 2. 인지적 관점의 기본 입장

### 2.1 인지란 무엇인가?

인지란 사고 과정, 정보 처리 과정, 상황 해석 과정을 의미한다.

```text
인지 = 사고 과정 = 정보 처리 과정 = 상황 해석 과정
```

성격의 인지적 관점에서는 성격을 다음과 같이 이해한다.

```text
성격 = 개인이 세상을 지각하고 해석하는 반복적 방식
```

따라서 인간의 성격과 행동을 이해하려면 그 사람이 가진 다음 요소를 살펴보아야 한다.

| 요소    | 의미                         |
| ----- | -------------------------- |
| 신념    | 자신, 타인, 세상에 대한 믿음          |
| 기대    | 앞으로 어떤 일이 일어날 것이라고 예상하는 방식 |
| 해석 방식 | 사건에 의미를 부여하는 방식            |
| 사고 과정 | 정보를 처리하고 판단하는 방식           |
| 인지 도식 | 자기와 세상을 이해하는 기본 틀          |

인지적 관점은 무의식이나 과거 자체보다 **현재 상황을 어떻게 해석하는가**에 초점을 둔다.

## 3. 심리적 현실

### 3.1 객관적 현실과 심리적 현실

인지적 관점에서는 인간이 객관적 현실을 그대로 받아들이는 것이 아니라, 자신의 경험과 신념을 통해 해석한 **심리적 현실**을 경험한다고 본다.

같은 사건을 보더라도 사람마다 다르게 느끼는 이유가 여기에 있다.

예를 들어 아이들이 분수대에서 노는 장면을 보고 어떤 사람은 “행복해 보인다”, “시원하겠다”고 해석할 수 있다. 반면 다른 사람은 “감기 걸리겠다”, “아프면 일정이 밀린다”고 해석할 수 있다.

같은 사건이지만 해석이 다르면 감정과 행동도 달라진다.

```text
같은 사건
→ 서로 다른 해석
→ 서로 다른 감정과 행동
```

### 3.2 Kelly의 개인 구성개념

Kelly는 인간이 세상을 이해할 때 각자 나름의 **개인 구성개념**을 사용한다고 보았다.

강의에서는 이를 **렌즈**에 비유했다.

```text
개인 구성개념 = 세상을 바라보는 렌즈
```

사람들은 같은 사건을 보더라도 서로 다른 렌즈를 통해 해석한다. 그래서 어떤 사람과 “잘 맞는다”고 느끼는 것은 세상을 해석하는 구성개념이 비슷하기 때문일 수 있다.

## 4. 인지적 관점의 주요 학자

10주차에서 다룬 인지적 관점의 핵심 학자는 다음 세 명이다.

| 학자    | 핵심 이론        | 주요 내용                           |
| ----- | ------------ | ------------------------------- |
| Kelly | 개인 구성개념 이론   | 인간은 자신만의 렌즈로 사건을 해석한다           |
| Ellis | ABC 모델, REBT | 사건 자체보다 사건에 대한 신념이 정서와 행동을 결정한다 |
| Beck  | 인지치료, 인지적 오류 | 역기능적 도식과 인지 왜곡이 심리적 문제를 유발한다    |

## 5. Ellis의 ABC 모델

### 5.1 ABC 모델의 구조

Ellis는 인간의 정서와 행동을 설명하기 위해 **ABC 모델**을 제시했다.

| 단계 | 의미               | 설명                |
| -- | ---------------- | ----------------- |
| A  | Activating Event | 촉발 사건, 선행 사건      |
| B  | Belief           | 사건에 대한 신념, 해석, 평가 |
| C  | Consequence      | 정서적·행동적 결과        |

핵심은 다음과 같다.

```text
A가 직접 C를 만드는 것이 아니라,
B가 A와 C 사이를 매개한다.
```

즉, 사람들은 사건 자체 때문에 고통받는 것이 아니라, 사건에 대한 자신의 생각과 신념 때문에 고통받는다.

### 5.2 ABC 모델 예시

```text
A: 친구가 카톡을 읽고 답장하지 않음
B: “나를 무시하는 게 분명해.”
C: 분노, 서운함, 관계 회피
```

하지만 B가 달라지면 C도 달라질 수 있다.

```text
A: 친구가 카톡을 읽고 답장하지 않음
B: “바쁘거나 몸이 안 좋을 수도 있어.”
C: 기다림, 비교적 안정된 감정
```

즉, 사건 자체보다 **사건에 대한 해석**이 정서와 행동을 결정한다.

## 6. 자동적 사고

### 6.1 자동적 사고의 의미

자동적 사고는 어떤 상황을 마주했을 때 매우 빠르게 떠오르는 생각이다.

강의에서는 자동적 사고가 너무 빠르게 지나가기 때문에 자신이 어떤 해석을 했는지 알아차리기 어렵다고 설명했다.

```text
사건 발생
→ 자동적 해석
→ 감정과 행동
```

예를 들어 누군가 나를 보며 웃을 때 다음과 같은 생각이 자동으로 떠오를 수 있다.

```text
“나를 비웃는 건가?”
“내가 뭔가 실수했나?”
```

하지만 실제로는 그 사람이 다른 이유로 웃고 있었을 수도 있다. 인지적 관점은 이런 빠른 해석이 실제 현실과 일치하는지 점검하려 한다.

## 7. Ellis가 본 인간

Ellis는 인간을 **합리적이면서 동시에 비합리적인 존재**로 보았다.

인간에게는 성장 가능성과 합리적 사고 능력이 있지만, 동시에 스스로를 괴롭히는 방식으로 비합리적으로 생각하는 경향도 있다.

특히 자신이 원하는 것을 얻지 못할 때 인간은 다음을 비난하는 경향이 있다.

* 자신
* 타인
* 세상

이러한 사고는 **자기 패배적 사고**로 이어질 수 있다.

```text
원하는 결과를 얻지 못함
→ 자신, 타인, 세상을 비난함
→ 자기 패배적 사고
→ 부적응적 감정과 행동
```

## 8. 합리적 사고와 비합리적 사고

### 8.1 합리적 사고의 의미

Ellis가 말한 합리적 사고는 도덕적으로 완벽하거나 항상 긍정적인 생각을 의미하지 않는다.

합리적 사고의 기준은 다음과 같다.

| 기준     | 질문                       |
| ------ | ------------------------ |
| 현실성    | 이 생각은 현실과 맞는가?           |
| 실용성    | 이 생각은 문제 해결에 도움이 되는가?    |
| 융통성    | 이 생각은 유연한가?              |
| 정서적 안정 | 이 생각은 감정을 지나치게 압도하지 않는가? |
| 기능성    | 이 생각은 내가 나답게 기능하도록 돕는가?  |

핵심 질문은 다음과 같다.

```text
“이 생각이 옳은가?”보다
“이 생각이 나에게 도움이 되는가?”
```

### 8.2 부정적 감정 자체가 문제는 아니다

REBT는 슬픔, 걱정, 긴장, 아쉬움, 분노 같은 감정을 모두 없애려는 접근이 아니다.

일상적 부정 감정은 자연스러운 감정이다.

문제가 되는 것은 감정이 지나치게 강해져서 개인이 기능하지 못하게 되는 경우이다.

| 구분         | 설명                            |
| ---------- | ----------------------------- |
| 건강한 부정 감정  | 아쉬움, 적절한 긴장, 유감, 일반적 슬픔       |
| 부적응적 정서 혼란 | 압도적 분노, 극심한 불안, 절망, 회피, 기능 저하 |

## 9. 비합리적 신념

### 9.1 비합리적 신념의 의미

비합리적 신념은 현실에서 충족되기 어려운 절대적이고 경직된 기대와 요구를 말한다.

주로 다음과 같은 표현으로 나타난다.

```text
반드시 ~해야 한다
꼭 그래야 한다
항상 그래야 한다
절대 ~하면 안 된다
그래야만 한다
```

이러한 사고는 **당위적 사고**, **절대적 사고**, **완벽주의적 사고**의 형태를 띤다.

### 9.2 세 가지 당위성

비합리적 신념은 크게 세 방향으로 나타난다.

| 구분         | 설명                        | 예시                   |
| ---------- | ------------------------- | -------------------- |
| 자신에 대한 당위성 | 내가 반드시 완벽해야 한다는 요구        | “나는 항상 인정받아야 한다.”    |
| 타인에 대한 당위성 | 타인이 반드시 나에게 잘해야 한다는 요구    | “친구라면 항상 내 편이어야 한다.” |
| 조건에 대한 당위성 | 세상이 반드시 내 기대대로 되어야 한다는 요구 | “사회는 항상 공정해야 한다.”    |

### 9.3 비합리적 신념의 대표 형태

| 유형        | 설명                 | 예시                            |
| --------- | ------------------ | ----------------------------- |
| 절대적 당위    | 반드시 그래야 한다는 사고     | “나는 절대 실수하면 안 된다.”            |
| 파국화       | 사건을 끔찍하게 과장함       | “이건 완전히 끝장이다.”                |
| 낮은 좌절 인내력 | 불편함을 견딜 수 없다고 생각함  | “이런 일은 도저히 못 참겠다.”            |
| 자기·타인 질책  | 행동이 아니라 인격 전체를 비난함 | “나는 무가치하다.”, “그 사람은 나쁜 인간이다.” |

## 10. REBT

### 10.1 REBT란?

REBT는 **Rational Emotive Behavior Therapy**의 약자로, 한국어로는 **합리적 정서행동치료** 또는 **인지·정서·행동치료**라고 한다.

Ellis가 만든 치료 접근으로, 비합리적 신념을 찾아내고 논박하여 더 합리적인 신념으로 바꾸는 것을 목표로 한다.

```text
REBT의 핵심:
비합리적 신념을 논박하여
더 현실적이고 유연하며 실용적인 신념으로 변화시키는 것
```

### 10.2 ABCDEF 모델

ABC 모델은 치료 과정에서 ABCDEF 모델로 확장된다.

| 단계 | 의미                    | 설명         |
| -- | --------------------- | ---------- |
| A  | Activating Event      | 촉발 사건      |
| B  | Belief                | 비합리적 신념    |
| C  | Consequence           | 정서적·행동적 결과 |
| D  | Dispute               | 비합리적 신념 논박 |
| E  | Effect                | 효과적인 새 인지  |
| F  | Feeling / New Feeling | 새로운 감정과 행동 |

핵심은 **D, 논박**이다.

논박은 상대를 비난하거나 생각이 틀렸다고 단정하는 것이 아니라, 그 생각이 현실적인지, 실용적인지, 융통성 있는지 따뜻하게 검토하는 과정이다.

## 11. REBT의 논박 기준

비합리적 신념을 논박할 때는 다음 기준을 사용할 수 있다.

### 11.1 현실성 논박

```text
정말 항상 그래야만 하는가?
그 생각을 뒷받침하는 증거가 있는가?
예외는 없는가?
```

### 11.2 실용성 논박

```text
이 생각이 문제 해결에 도움이 되는가?
이 생각이 나를 더 기능하게 만드는가?
이 생각이 내 목표 달성에 도움이 되는가?
```

### 11.3 융통성 논박

```text
다른 해석은 가능한가?
다른 대안은 없는가?
이것만이 유일한 길인가?
```

### 11.4 자기수용적 논박

```text
실수했다고 해서 나 전체가 무가치한가?
한 번의 실패가 내 전체 능력을 결정하는가?
```

## 12. 초등학교 5학년 숙제 사례

강의에서는 영어 숙제가 너무 많아 힘들어하는 초등학교 5학년 사례를 통해 논박을 설명했다.

아이의 사고는 다음과 같았다.

```text
“숙제는 무조건 다 해야 해.”
“못 하면 큰일 나.”
“선생님께 혼날 거야.”
```

이 사고를 현실성, 실용성, 융통성으로 논박할 수 있다.

| 기준  | 논박                                     |
| --- | -------------------------------------- |
| 현실성 | 모든 숙제를 항상 완벽하게 해야 하는 것은 아니다            |
| 실용성 | 목표는 숙제를 완벽히 끝내는 것만이 아니라 영어 실력을 키우는 것이다 |
| 융통성 | 오늘 중요한 것부터 하고 부족한 것은 내일 이어서 할 수 있다     |

이 사례에서 중요한 점은 비합리적 사고가 당사자에게는 매우 진지하고 절대적인 진실처럼 느껴진다는 것이다.

그래서 논박은 차갑게 지적하는 방식이 아니라, 따뜻하고 과학적인 검토 방식이어야 한다.

## 13. AFROG 사고 점검 질문

Beck과 Emery가 제시한 사고 점검 기준으로 AFROG가 소개되었다.

| 기준         | 질문                         |
| ---------- | -------------------------- |
| A, Alive   | 이 사고는 나를 생기 있게 하는가?        |
| F, Feel    | 이 사고의 결과로 기분이 더 나아졌는가?     |
| R, Reality | 이 사고는 현실적인가?               |
| O, Other   | 이 사고는 다른 사람과의 관계에 도움이 되는가? |
| G, Goals   | 이 사고는 나의 목표 성취에 도움이 되는가?   |

이 기준은 자신의 자동적 사고가 합리적인지 점검하는 데 유용하다.

## 14. Ellis의 11가지 비합리적 신념

Ellis는 인간이 자주 가지는 비합리적 신념을 여러 유형으로 정리했다.

강의에서는 모두 암기할 필요는 없다고 언급되었지만, 주요 방향은 다음과 같다.

| 유형        | 설명                               |
| --------- | -------------------------------- |
| 타인 인정 욕구  | 모든 사람에게 인정받아야 한다는 신념             |
| 과대한 자기 기대 | 자신이 항상 유능하고 성공적이어야 한다는 신념        |
| 비난 경향성    | 잘못한 사람은 반드시 비난받아야 한다는 신념         |
| 파국화       | 사소한 좌절을 끔찍한 결과로 해석하는 신념          |
| 회피        | 어려운 문제는 피하는 것이 낫다는 신념            |
| 의존        | 강한 타인에게 의존해야 한다는 신념              |
| 지나친 타인 염려 | 타인의 문제까지 과도하게 책임지려는 신념           |
| 완벽주의      | 모든 문제에는 완벽하고 정확한 해결책이 있어야 한다는 신념 |

특히 완벽주의는 단순히 “내가 완벽해야 한다”뿐 아니라, “모든 문제에는 완벽한 해결책이 존재한다”는 사고와도 관련된다.

## 15. 비합리적 신념 측정의 문제

Ellis의 비합리적 신념을 측정하는 기존 도구들은 여러 한계를 가진다.

주요 문제는 다음과 같다.

| 문제        | 설명                                   |
| --------- | ------------------------------------ |
| 하위요인 불명확  | 비합리적 신념의 핵심 요소가 명확히 구분되지 않음          |
| 문항 중복     | 여러 개념이 하나의 문항 안에 섞여 있음               |
| 사고와 결과 혼동 | 신념이 아니라 감정이나 행동 결과를 측정하는 경우가 있음      |
| 특정 상황 편향  | 특정 장면에 대한 반응을 일반 신념처럼 측정할 수 있음       |
| 자기보고 한계   | 응답자가 자신의 실제 자동적 사고를 정확히 인식하지 못할 수 있음 |

따라서 비합리적 신념을 더 정확히 파악하려면 **구체적 상황에서 떠오르는 자동적 사고를 확인하고**, 그 사고의 논리성, 현실성, 융통성, 실용성을 평가하는 방식이 필요하다.

```text
상황 제시
→ 자동적 사고 확인
→ 비합리적 신념 분석
→ 논리성·현실성·융통성·실용성 평가
```

## 17. 사회불안과 인지적 관점

### 17.1 사회불안의 의미

사회불안은 사회적 상황에서 과도한 두려움을 느끼고, 타인의 부정적 평가를 지나치게 걱정하는 상태이다.

사회불안을 경험하는 사람은 다음과 같은 특징을 보인다.

* 자신의 사회적 능력을 과소평가함
* 타인의 부정적 평가 가능성을 과대평가함
* 모호한 피드백도 부정적으로 해석함
* 실수에 대해 과도하게 걱정함
* 사회적 상황을 회피함
* 회피로 인해 사회적 기술 학습 기회가 줄어듦

### 17.2 사회불안의 인지적 구조

```text
사회적 상황
→ “사람들이 나를 이상하게 볼 것이다”
→ 불안과 수치심
→ 회피
→ 사회적 경험 부족
→ 자기효능감 저하
→ 사회불안 유지
```

REBT는 사회불안의 핵심 원인을 타인의 평가와 실수에 대한 비합리적 신념에서 찾는다.

## 18. REBT와 심리극의 절충

### 18.1 심리극의 의미

심리극은 Moreno가 창안한 집단 심리치료 방법으로, 말로만 문제를 다루는 것이 아니라 내담자의 경험을 무대 위에서 행위로 재현한다.

| 구성요소  | 설명                      |
| ----- | ----------------------- |
| 주인공   | 자신의 문제와 갈등을 표현하는 중심 인물  |
| 디렉터   | 치료 과정을 이끄는 진행자          |
| 보조 자아 | 주인공의 중요한 타인이나 내면 일부를 표현 |
| 관객    | 관찰하고 피드백을 제공하는 집단원      |
| 무대    | 내면세계와 현실을 연결하는 공간       |

심리극의 주요 단계는 다음과 같다.

| 단계  | 설명                       |
| --- | ------------------------ |
| 준비  | 집단 분위기 형성, 불안 감소, 주인공 선정 |
| 행위화 | 갈등 상황을 무대 위에서 재현         |
| 나누기 | 통찰과 감정을 공유하고 일상 적용을 계획   |

### 18.2 주요 기법

| 기법    | 설명                      |
| ----- | ----------------------- |
| 역할 교대 | 상대방의 입장이 되어 상황을 경험함     |
| 이중 자아 | 표현하지 못한 내면의 감정과 생각을 드러냄 |
| 거울 기법 | 자신의 행동을 밖에서 객관적으로 관찰함   |

### 18.3 REBT와 심리극의 상호보완성

| REBT             | 심리극                    |
| ---------------- | ---------------------- |
| 비합리적 신념을 찾고 논박함  | 정서와 행동을 실제 장면에서 경험함    |
| ABCDE 모델로 구조화 가능 | 역할 교대, 이중 자아, 거울 기법 활용 |
| 인지적 재구성에 강함      | 정서 안정화와 행동 연습에 강함      |
| 자기수용과 불편감 감내 훈련  | 사회적 자기효능감 향상           |

사회불안 치료에서 REBT는 신념을 바꾸는 데 강점이 있고, 심리극은 실제 사회적 상황을 안전하게 재현하고 연습하는 데 강점이 있다.

두 접근을 결합하면 사회불안을 **인지·정서·행동 차원에서 통합적으로** 다룰 수 있다.

## 19. 자동적 사고 점검 방법

자신의 자동적 사고를 점검할 때는 다음 질문을 활용할 수 있다.

```text
이 생각에는 사실적 근거가 있는가?
이 생각은 현실적인가?
이 생각은 내 삶에 도움이 되는가?
이 생각 때문에 기분과 관계가 나빠지는 것은 아닌가?
이 생각은 내 목표 달성에 도움이 되는가?
다른 해석은 가능한가?
```

목표는 비합리적 사고를 완전히 없애는 것이 아니다.

강의에서는 이를 다음처럼 설명했다.

```text
100% 확신하던 생각을
90%, 80%, 70% 정도로 낮추는 것
```

즉, 굳어 있는 생각을 조금 더 유연하고 말랑하게 만드는 것이 핵심이다.

## 20. 최종 핵심 정리

10주차의 핵심은 다음과 같다.

```text
인간의 감정과 행동은 사건 자체가 아니라,
그 사건에 대한 해석과 신념에 의해 결정된다.
```

인지적 관점은 인간이 객관적 현실에 그대로 반응하는 존재가 아니라, 각자의 구성개념과 신념을 통해 현실을 해석하는 존재라고 본다.

Ellis의 ABC 모델은 사건과 결과 사이에 신념이 있음을 설명한다.

비합리적 신념은 “반드시”, “항상”, “절대”, “그래야만 한다”와 같은 당위적 사고로 나타나며, 불안, 우울, 분노, 회피 행동을 유발할 수 있다.

REBT는 이러한 비합리적 신념을 현실성, 실용성, 융통성 기준으로 논박하여 더 합리적이고 기능적인 신념으로 바꾸는 치료 접근이다.

Beck은 이분법적 사고, 선택적 추론, 파국화, 과잉일반화, 개인화, 정서적 추론과 같은 인지적 오류를 제시했다.

사회불안은 타인의 평가에 대한 비합리적 신념과 회피 행동에 의해 유지될 수 있으며, REBT와 심리극을 결합하면 인지, 정서, 행동을 통합적으로 다룰 수 있다.

결론적으로 10주차의 핵심 문장은 다음과 같다.

```text
성격은 개인이 세상을 해석하는 반복적 사고 방식이며,
그 사고 방식이 정서와 행동을 결정한다.
```
