## 2. 복습 실습

## 2.1 구구단 4단까지 출력

### 목표

`2단`부터 `4단`까지만 콘솔에 출력한다.

### 핵심 개념

- 중첩 반복문
- `break`
- 조건문
- 콘솔 출력

### 예시 코드

```js
for (let i = 2; i <= 9; i++) {
  if (i > 4) {
    break;
  }

  for (let j = 1; j <= 9; j++) {
    console.log(`${i} x ${j} = ${i * j}`);
  }

  console.log('---');
}
````

### 설명

* 바깥쪽 `for`문은 2단부터 9단까지 반복한다.
* `i > 4`가 되는 순간 `break`가 실행된다.
* 따라서 5단부터는 출력되지 않는다.
* 안쪽 `for`문은 각 단의 `1~9` 곱셈 결과를 출력한다.

### 참고

```js
for (let i = 2; i <= 4; i++)
```

처럼 반복 조건을 직접 `4`까지로 제한해도 같은 결과를 얻을 수 있다.

## 2.2 배열에서 세 글자 단어만 출력

### 목표

과일 이름 배열에서 글자 수가 3인 요소만 출력한다.

### 예시 코드

```js
const fruits = ['사과', '바나나', '포도', '오렌지', '딸기', '키위'];

for (let i = 0; i < fruits.length; i++) {
  if (fruits[i].length === 3) {
    console.log(fruits[i]);
  }
}
```

### 출력 결과

```txt
바나나
오렌지
```

### 핵심 개념

* `fruits.length`

  * 배열의 길이
  * 예: 배열 요소 개수 6개

* `fruits[i].length`

  * 문자열의 길이
  * 예: `"바나나"`의 길이 3

### 주의할 점

같은 `length`이지만 대상이 다르다.

```js
fruits.length       // 배열 길이
fruits[i].length    // 문자열 길이
```

# 3. JavaScript와 객체

## 3.1 JavaScript는 객체 기반 언어

JavaScript는 객체 기반 언어이다.
원시값을 제외한 대부분의 값은 객체로 취급된다.

## 3.2 원시 타입

JavaScript의 원시 타입은 다음 7가지이다.

| 타입          | 예시                  |
| ----------- | ------------------- |
| `string`    | `'안녕'`, `"hello"`   |
| `number`    | `42`, `3.14`        |
| `boolean`   | `true`, `false`     |
| `null`      | `null`              |
| `undefined` | `undefined`         |
| `symbol`    | `Symbol()`          |
| `bigint`    | `9007199254740992n` |

## 3.3 객체 타입

| 종류     | 예시                   |
| ------ | -------------------- |
| 일반 객체  | `{ name: '홍길동' }`    |
| 배열     | `[1, 2, 3]`          |
| 함수     | `function() {}`      |
| 내장 객체  | `Math`, `Date`       |
| DOM 객체 | `document`, `window` |

# 4. Symbol과 BigInt

## 4.1 Symbol

`Symbol`은 절대 중복되지 않는 고유한 값을 만들 때 사용한다.

### 특징

* 같은 설명을 넣어도 서로 다른 값이다.
* 객체의 숨겨진 key처럼 사용할 수 있다.
* 일반적인 `for...in` 순회에서 보이지 않는다.

### 예시

```js
const s1 = Symbol('id');
const s2 = Symbol('id');

console.log(s1 === s2); // false
```

### 객체 key로 사용

```js
const key = Symbol('secret');
const obj = {};

obj[key] = '비밀 데이터';
obj.name = '공개 데이터';

for (const k in obj) {
  console.log(k); // name만 출력
}

console.log(obj[key]); // 비밀 데이터
```

---

## 4.2 BigInt

`BigInt`는 매우 큰 정수를 정확하게 다루기 위한 타입이다.

### 특징

* 숫자 뒤에 `n`을 붙인다.
* 일반 `number` 타입보다 더 큰 정수를 정확히 표현할 수 있다.
* 암호학, 정밀 금융 계산 등에서 사용될 수 있다.

### 예시

```js
const big = 9007199254740992n;
```

# 5. 거의 모든 것이 객체

## 5.1 배열도 객체

```js
const arr = [1, 2, 3];

arr.length;  // 속성
arr.push(4); // 메서드

typeof arr;  // "object"
```

## 5.2 함수도 객체

```js
function hello() {}

hello.name;
hello.length;
```

## 5.3 문자열도 객체처럼 동작

문자열은 원시값이지만 객체처럼 속성과 메서드를 사용할 수 있다.

```js
const s = 'hello';

s.length;        // 5
s.toUpperCase(); // 'HELLO'
```

## 5.4 이미 사용한 객체 예시

| 코드                   | 객체 여부            |
| -------------------- | ---------------- |
| `console.log()`      | 객체의 메서드          |
| `prompt()`           | `window` 객체의 메서드 |
| `Math.random()`      | `Math` 객체의 메서드   |
| `[1, 2, 3]`          | 배열 객체            |
| `arr.push()`         | 배열 객체의 메서드       |
| `'hi'.toUpperCase()` | 문자열 메서드          |
| `function() {}`      | 함수 객체            |

# 6. 객체란?

## 6.1 객체의 정의

객체는 데이터와 기능을 하나로 묶은 집합이다.

객체는 다음 두 가지로 구성된다.

* 속성

  * 객체가 가진 데이터
* 메서드

  * 객체 안에 들어 있는 함수

## 6.2 key와 value

객체의 각 요소는 `key: value` 형태로 표현된다.

```js
const student = {
  name: '홍길동',
  age: 21,
  major: '소프트웨어학부',

  birthday() {
    this.age += 1;
    console.log(`이제 ${this.age}살`);
  },

  introduce() {
    console.log(`저는 ${this.name}입니다`);
  }
};
```

## 6.3 속성과 메서드

| 구분  | key         | value              |
| --- | ----------- | ------------------ |
| 속성  | `name`      | `'홍길동'`            |
| 속성  | `age`       | `21`               |
| 속성  | `major`     | `'소프트웨어학부'`        |
| 메서드 | `birthday`  | `function() {...}` |
| 메서드 | `introduce` | `function() {...}` |

## 6.4 객체 사용 예시

```js
student.name;
student.introduce();
student.birthday();
```

# 7. 객체 생성 방법

## 7.1 객체 리터럴

가장 많이 사용하는 방식이다.

```js
const student1 = {
  name: '홍길동',
  age: 21
};
```

## 7.2 `new Object()`

빈 객체를 만든 뒤 속성을 추가하는 방식이다.

```js
const student2 = new Object();

student2.name = '홍길동';
student2.age = 21;
```

## 7.3 빈 객체 리터럴

```js
const student3 = {};

student3.name = '홍길동';
student3.age = 21;
```

## 7.4 정리

| 방법             | 특징                |
| -------------- | ----------------- |
| `{}`           | 간결하고 가장 많이 사용     |
| `new Object()` | 전통적 방식, 잘 사용하지 않음 |
| 빈 리터럴 `{}`     | 동적으로 속성 추가 가능     |

### 실무 권장

객체 생성은 보통 리터럴 방식 `{}`을 사용한다.

# 8. 배열 생성 방법

## 8.1 배열 리터럴

```js
const arr1 = [1, 2, 3];
const arr2 = ['a', 'b', 'c'];
const arr3 = [];
```

## 8.2 `new Array()`

```js
const arr4 = new Array(1, 2, 3);
```

## 8.3 주의할 점

```js
const arr5 = new Array(3);
```

이 코드는 `[3]`이 아니라 길이가 3인 빈 배열을 만든다.

```js
[3];          // 요소가 3 하나인 배열
new Array(3); // 길이가 3인 빈 배열
```

## 8.4 정리

| 코드                   | 결과         |
| -------------------- | ---------- |
| `[]`                 | 빈 배열       |
| `[1, 2, 3]`          | 요소 3개 배열   |
| `new Array()`        | 빈 배열       |
| `new Array(3)`       | 길이 3인 빈 배열 |
| `new Array(1, 2, 3)` | 요소 3개 배열   |

### 실무 권장

배열 생성은 리터럴 방식 `[]`을 사용하는 것이 좋다.

# 9. 객체 기본 사용

객체는 생성 후에도 속성을 자유롭게 읽고, 추가하고, 변경하고, 삭제할 수 있다.

## 9.1 예시 객체

```js
const student = {
  name: '홍길동',
  age: 21
};
```

## 9.2 읽기

```js
console.log(student.name);
console.log(student.age);
console.log(student.email); // undefined
```

없는 속성에 접근하면 오류가 아니라 `undefined`가 반환된다.

## 9.3 변경

```js
student.age = 22;
student.name = '김철수';
```

## 9.4 추가

```js
student.email = 'kim@kookmin.ac.kr';
student.major = '소프트웨어학부';
```

## 9.5 삭제

```js
delete student.age;
```

## 9.6 CRUD 정리

| 동작 | 문법               |
| -- | ---------------- |
| 읽기 | `obj.key`        |
| 변경 | `obj.key = 값`    |
| 추가 | `obj.key = 값`    |
| 삭제 | `delete obj.key` |

## 9.7 `const`와 객체

`const`로 선언한 객체도 내부 속성은 변경할 수 있다.

```js
const student = { age: 21 };

student.age = 22; // 가능
student = {};     // 불가능
```

`const`는 변수 재할당을 막는 것이지, 객체 내부 변경을 막는 것이 아니다.

# 10. 속성 접근 방법

객체의 속성에 접근하는 방법은 두 가지가 있다.

* 점 표기법
* 대괄호 표기법

## 10.1 점 표기법

```js
student.name;
student.age;
```

### 특징

* 간단하고 가독성이 좋다.
* 일반적인 속성 접근에 사용한다.
* 공백이나 특수문자가 있는 key에는 사용할 수 없다.
* 변수에 저장된 key를 사용할 수 없다.

## 10.2 대괄호 표기법

```js
student['name'];
student['home address'];
```

### 특징

* 공백이 있는 key에 접근 가능하다.
* 특수문자가 있는 key에 접근 가능하다.
* 변수로 key를 지정할 수 있다.

## 10.3 변수 key 접근

```js
const key = 'age';

student[key]; // student['age']와 같음
```

## 10.4 차이점

```js
student.key;
```

위 코드는 `"key"`라는 이름의 속성을 찾는다.

```js
student[key];
```

위 코드는 변수 `key`에 들어 있는 값으로 속성을 찾는다.

## 10.5 정리

| 항목       | 점 표기법     | 대괄호 표기법      |
| -------- | --------- | ------------ |
| 문법       | `obj.key` | `obj['key']` |
| 가독성      | 좋음        | 보통           |
| 공백 key   | 불가        | 가능           |
| 특수문자 key | 불가        | 가능           |
| 변수 key   | 불가        | 가능           |

### 실무 가이드

* 평소에는 점 표기법 사용
* key가 동적이거나 공백/특수문자가 있으면 대괄호 표기법 사용

# 11. 객체 vs 배열

## 11.1 배열도 객체

배열은 객체의 한 종류이다.

```js
typeof []; // "object"
```

하지만 일반 객체와 배열은 용도가 다르다.

## 11.2 배열

배열은 같은 종류의 데이터를 순서대로 저장할 때 사용한다.

```js
const scores = [85, 92, 78];

scores[0]; // 85
```

## 11.3 객체

객체는 하나의 대상이 가진 여러 속성을 표현할 때 사용한다.

```js
const student = {
  name: '홍길동',
  age: 21,
  scores: [85, 92, 78]
};
```

## 11.4 객체의 배열

실무에서 가장 흔한 데이터 구조이다.

```js
const students = [
  { name: '홍길동', age: 21 },
  { name: '김철수', age: 22 },
  { name: '이영희', age: 20 }
];
```

### 접근 예시

```js
students[0].name; // 홍길동
students[1].age;  // 22
```

## 11.5 객체와 배열 비교

| 항목       | 객체 `{}`    | 배열 `[]`    |
| -------- | ---------- | ---------- |
| 접근 방식    | key 이름     | 인덱스        |
| 순서       | 중요하지 않음    | 중요함        |
| 용도       | 하나의 대상 표현  | 같은 종류의 목록  |
| `typeof` | `"object"` | `"object"` |
| `length` | 없음         | 있음         |

## 11.6 배열 확인

배열인지 확인할 때는 `typeof`가 아니라 `Array.isArray()`를 사용한다.

```js
Array.isArray(students);
```

# 12. 객체 순회

객체의 모든 key와 value를 순회할 때는 다음 방법을 사용할 수 있다.

## 12.1 `for...in`

객체의 key를 순회한다.

```js
const student = {
  name: '홍길동',
  age: 21,
  major: '소프트웨어학부'
};

for (const key in student) {
  console.log(key, student[key]);
}
```

## 12.2 `Object.keys()`

객체의 모든 key를 배열로 반환한다.

```js
Object.keys(student);
// ['name', 'age', 'major']
```

## 12.3 `Object.values()`

객체의 모든 value를 배열로 반환한다.

```js
Object.values(student);
// ['홍길동', 21, '소프트웨어학부']
```

## 12.4 `Object.entries()`

객체의 key-value 쌍을 배열로 반환한다.

```js
Object.entries(student);
// [['name', '홍길동'], ['age', 21], ['major', '소프트웨어학부']]
```

## 12.5 정리

| 방법                 | 반환                |
| ------------------ | ----------------- |
| `for...in`         | 각 key             |
| `Object.keys()`    | key 배열            |
| `Object.values()`  | value 배열          |
| `Object.entries()` | `[key, value]` 배열 |

## 12.6 `for...in` vs `for...of`

| 문법         | 용도         |
| ---------- | ---------- |
| `for...in` | 객체의 key 순회 |
| `for...of` | 배열의 값 순회   |

```js
for (const key in obj) {}

for (const value of arr) {}
```

# 13. 객체 메서드와 this

## 13.1 메서드

객체의 value에 함수를 넣은 것을 메서드라고 한다.

```js
const student = {
  name: '홍길동',
  age: 21,

  greet: function() {
    console.log('안녕하세요');
  },

  introduce() {
    console.log('저는 학생입니다');
  }
};
```

## 13.2 메서드 호출

```js
student.greet();
student.introduce();
```

메서드는 함수처럼 `()`를 붙여 호출한다.

## 13.3 this

메서드 안에서 자기 객체의 속성에 접근할 때 `this`를 사용한다.

```js
const student = {
  name: '홍길동',
  age: 21,

  sayHello() {
    console.log(`안녕, 나는 ${this.name}`);
    console.log(`나이는 ${this.age}살`);
  }
};
```

## 13.4 this의 의미

```js
student.sayHello();
```

이렇게 호출하면 `sayHello()` 안의 `this`는 `student` 객체를 가리킨다.

## 13.5 메서드에서 다른 메서드 호출

```js
const student = {
  greet() {
    console.log('안녕하세요');
  },

  sayHello() {
    console.log('안녕');
  },

  showAll() {
    this.greet();
    this.sayHello();
  }
};
```

## 13.6 함수와 메서드 차이

| 구분  | 설명         | 예시                |
| --- | ---------- | ----------------- |
| 함수  | 독립적으로 호출   | `hello()`         |
| 메서드 | 객체에 소속된 함수 | `student.hello()` |

# 14. 전역 객체 window

## 14.1 window

브라우저에는 최상위 객체인 `window`가 있다.

```js
window.alert('안녕');
window.console.log('hi');
window.document.querySelector('h1');
```

## 14.2 window 생략

`window.`은 생략할 수 있다.

```js
alert('안녕');                 // window.alert()
prompt('이름?');               // window.prompt()
confirm('확실해?');            // window.confirm()
console.log('hi');             // window.console.log()
document.querySelector('h1');  // window.document.querySelector()
```

## 14.3 전역 변수와 window

`var`로 선언한 전역 변수는 `window` 객체의 속성이 된다.

```js
var myName = '홍길동';

console.log(window.myName); // 홍길동
```

단, `let`과 `const`는 `window`에 붙지 않는다.

# 16. String 객체 메서드

문자열은 원시값이지만 String 객체의 메서드를 사용할 수 있다.

## 16.1 주요 메서드

| 메서드                | 설명            | 예시                                     |
| ------------------ | ------------- | -------------------------------------- |
| `length`           | 문자열 길이        | `'hello'.length` → `5`                 |
| `toUpperCase()`    | 대문자 변환        | `'hi'.toUpperCase()` → `'HI'`          |
| `toLowerCase()`    | 소문자 변환        | `'HI'.toLowerCase()` → `'hi'`          |
| `charAt(n)`        | n번째 글자        | `'abc'.charAt(1)` → `'b'`              |
| `indexOf('x')`     | x의 위치         | `'abc'.indexOf('b')` → `1`             |
| `includes('x')`    | 포함 여부         | `'abc'.includes('b')` → `true`         |
| `slice(a, b)`      | a부터 b-1까지 자르기 | `'hello'.slice(1, 4)` → `'ell'`        |
| `substring(a, b)`  | slice와 비슷     | `'hello'.substring(1, 4)` → `'ell'`    |
| `split('x')`       | x 기준 배열 분리    | `'a,b,c'.split(',')` → `['a','b','c']` |
| `replace('a','b')` | 문자열 변경        | `'cat'.replace('c','b')` → `'bat'`     |
| `trim()`           | 양쪽 공백 제거      | `' hi '.trim()` → `'hi'`               |
| `repeat(n)`        | n번 반복         | `'ab'.repeat(3)` → `'ababab'`          |
| `startsWith('x')`  | x로 시작하는지 확인   | `'abc'.startsWith('a')` → `true`       |
| `endsWith('x')`    | x로 끝나는지 확인    | `'abc'.endsWith('c')` → `true`         |

## 16.2 활용 예시

```js
const email = '  HONG@kookmin.ac.kr  ';

email.trim().toLowerCase();
// 'hong@kookmin.ac.kr'

email.trim().split('@')[1];
// 'kookmin.ac.kr'

email.includes('@');
// true
```

## 16.3 문자열은 불변

문자열 메서드는 원본 문자열을 직접 바꾸지 않고 새 문자열을 반환한다.

```js
s = s.toUpperCase();
```

처럼 재할당해야 변경된 값을 사용할 수 있다.

# 17. Array 객체 메서드

배열 메서드는 원본을 바꾸는 메서드와 새 배열을 반환하는 메서드로 구분해야 한다.

## 17.1 주요 메서드

| 메서드            | 설명                   | 원본 변경 |
| -------------- | -------------------- | ----- |
| `push(x)`      | 맨 뒤에 추가              | O     |
| `pop()`        | 맨 뒤 요소 제거            | O     |
| `unshift(x)`   | 맨 앞에 추가              | O     |
| `shift()`      | 맨 앞 요소 제거            | O     |
| `splice(i, n)` | i부터 n개 제거            | O     |
| `sort()`       | 정렬                   | O     |
| `reverse()`    | 뒤집기                  | O     |
| `slice(a, b)`  | a부터 b-1까지 잘라 새 배열 반환 | X     |
| `concat(arr)`  | 배열 합쳐 새 배열 반환        | X     |
| `join('x')`    | 구분자로 문자열 연결          | X     |
| `indexOf(x)`   | x의 위치                | X     |
| `includes(x)`  | 포함 여부                | X     |
| `forEach(fn)`  | 각 요소에 함수 실행          | X     |
| `map(fn)`      | 각 요소를 변환한 새 배열 반환    | X     |
| `filter(fn)`   | 조건에 맞는 요소만 새 배열 반환   | X     |
| `find(fn)`     | 조건에 맞는 첫 요소 반환       | X     |

## 17.2 활용 예시

```js
const nums = [3, 1, 4, 1, 5, 9, 2];

nums.sort();
// [1, 1, 2, 3, 4, 5, 9]

nums.join('-');
// '1-1-2-3-4-5-9'

nums.filter(n => n % 2 === 0);
// [2, 4]

nums.map(n => n * 2);
// [2, 2, 4, 6, 8, 10, 18]
```

## 17.3 주의할 점

* `push()`, `pop()`, `sort()` 등은 원본 배열을 바꾼다.
* `slice()`, `map()`, `filter()` 등은 새 배열을 반환한다.

# 18. Math 객체 메서드

`Math`는 수학 연산을 위한 객체이다.
`new` 없이 바로 사용한다.

## 18.1 주요 메서드와 속성

| 메서드/속성              | 설명           | 예시                      |
| ------------------- | ------------ | ----------------------- |
| `Math.PI`           | 원주율          | `3.141592653589793`     |
| `Math.abs(x)`       | 절댓값          | `Math.abs(-5)` → `5`    |
| `Math.floor(x)`     | 내림           | `Math.floor(3.7)` → `3` |
| `Math.ceil(x)`      | 올림           | `Math.ceil(3.1)` → `4`  |
| `Math.round(x)`     | 반올림          | `Math.round(3.5)` → `4` |
| `Math.max(a,b,...)` | 최댓값          | `Math.max(1,5,3)` → `5` |
| `Math.min(a,b,...)` | 최솟값          | `Math.min(1,5,3)` → `1` |
| `Math.pow(a,b)`     | a의 b제곱       | `Math.pow(2,3)` → `8`   |
| `Math.sqrt(x)`      | 제곱근          | `Math.sqrt(16)` → `4`   |
| `Math.random()`     | 0 이상 1 미만 난수 | `0.6273...`             |

## 18.2 1~10 사이 정수 난수

```js
Math.floor(Math.random() * 10) + 1;
```

### 동작 과정

```txt
Math.random()          → 0 이상 1 미만
Math.random() * 10     → 0 이상 10 미만
Math.floor(...)        → 0부터 9까지 정수
+ 1                    → 1부터 10까지 정수
```

## 18.3 배열 최댓값

```js
const nums = [3, 1, 4, 1, 5];

Math.max(...nums); // 5
```

## 18.4 소수점 둘째 자리까지 반올림

```js
Math.round(3.14159 * 100) / 100;
// 3.14
```

## 18.5 주의할 점

```js
new Math(); // 불가능
Math.floor(x); // 가능
```

`Math`는 생성자 함수로 사용하는 객체가 아니다.

# 20. Object 객체 메서드

`Object` 객체는 일반 객체를 다루기 위한 유틸리티 메서드를 제공한다.

## 20.1 주요 메서드

| 메서드                     | 설명                  | 반환               |
| ----------------------- | ------------------- | ---------------- |
| `Object.keys(obj)`      | 모든 key 반환           | 배열               |
| `Object.values(obj)`    | 모든 value 반환         | 배열               |
| `Object.entries(obj)`   | `[key, value]` 쌍 반환 | 배열               |
| `Object.assign(t, s)`   | s를 t에 복사            | 병합된 객체           |
| `Object.freeze(obj)`    | 객체 동결               | 원본 객체            |
| `Object.isFrozen(obj)`  | 동결 여부 확인            | `true` / `false` |
| `obj.hasOwnProperty(k)` | 속성 존재 여부 확인         | `true` / `false` |

## 20.2 객체 합치기

```js
const student = {
  name: '홍길동',
  age: 21
};

const extra = {
  major: '컴공'
};

Object.assign(student, extra);
```

결과:

```js
{
  name: '홍길동',
  age: 21,
  major: '컴공'
}
```

## 20.3 객체 동결

```js
Object.freeze(student);

student.age = 99;

console.log(student.age); // 기존 값 유지
```

`Object.freeze()`를 사용하면 객체 속성을 변경할 수 없다.

## 20.4 속성 존재 여부 확인

```js
student.hasOwnProperty('name');  // true
student.hasOwnProperty('email'); // false
```

# 21. 실습 1: 시스템 날짜 출력

## 21.1 목표

`Date` 객체를 사용하여 오늘 날짜와 현재 시간을 다양한 형식으로 콘솔에 출력한다.

## 21.2 예시 코드

```js
const now = new Date();

const year = now.getFullYear();
const month = now.getMonth() + 1;
const date = now.getDate();
const day = now.getDay();
const hours = now.getHours();
const mins = now.getMinutes();
const secs = now.getSeconds();

const days = ['일', '월', '화', '수', '목', '금', '토'];
const dayName = days[day];

console.log(`오늘은 ${year}년 ${month}월 ${date}일`);
console.log(`${dayName}요일`);
console.log(`현재 시각: ${hours}:${mins}:${secs}`);

console.log(
  `${year}-${month}-${date} (${dayName}) ${hours}:${mins}:${secs}`
);

console.log(now.toLocaleString());
```

## 21.3 핵심 포인트

* `getMonth()`는 0부터 시작하므로 `+ 1` 필요
* `getDay()`는 숫자로 요일을 반환하므로 배열을 활용해 문자로 변환
* `toLocaleString()`을 사용하면 지역 형식에 맞는 날짜/시간 문자열을 얻을 수 있음

# 22. 실습 2: 객체의 배열 다루기

## 22.1 목표

for (const s of students) {

# 23. 핵심 정리

## 23.1 객체

* 객체는 `key: value` 쌍의 집합이다.
* 객체 안의 데이터는 속성이라고 한다.
* 객체 안의 함수는 메서드라고 한다.
* 객체 속성은 추가, 변경, 삭제할 수 있다.
* `const` 객체도 내부 속성 변경은 가능하다.

## 23.2 속성 접근

* 일반적인 접근은 점 표기법을 사용한다.

```js
obj.key;
```

* 동적인 key, 공백 key, 특수문자 key는 대괄호 표기법을 사용한다.

```js
obj['key'];
obj[key];
```

## 23.3 객체와 배열

* 배열도 객체의 한 종류이다.
* 배열은 같은 종류의 목록을 저장할 때 사용한다.
* 객체는 하나의 대상이 가진 여러 속성을 표현할 때 사용한다.
* 실무에서는 객체의 배열 구조가 자주 사용된다.

## 23.4 객체 순회

```js
for (const key in obj) {}
Object.keys(obj);
Object.values(obj);
Object.entries(obj);
```

## 23.5 this

* 메서드 안에서 `this`는 그 메서드를 호출한 객체를 가리킨다.

```js
student.sayHello();
```

이 경우 `sayHello()` 내부의 `this`는 `student`이다.

## 23.6 window

* 브라우저의 전역 객체는 `window`이다.
* `alert()`, `prompt()`, `console.log()` 등은 사실 `window` 객체의 메서드 또는 속성이다.
* `window.`은 대부분 생략 가능하다.

## 23.7 내장 객체

* `String`
* `Array`
* `Math`
* `Date`
* `Object`
* `Number`
* `Boolean`

각 내장 객체는 자주 사용하는 메서드를 제공하므로 필요할 때 찾아서 활용하면 된다.
