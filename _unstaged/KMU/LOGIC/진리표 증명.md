# 진리표 증명

## 1. 핵심 주제

명제 논리에서 어떤 논증이 **타당한지 부당한지**를 판별하기 위해 **진리표**를 사용하는 방법을 다룬다.

다루는 방식은 크게 두 가지다.

1. **직접 진리표**
2. **간접 진리표**

## 2. 직접 진리표

### 2.1. 개념

직접 진리표는 논증에 등장하는 모든 단순명제의 가능한 진리값 조합을 전부 나열한 뒤,  
**전제가 모두 참인 경우에 결론이 거짓인 행이 있는지** 확인하는 방식이다.

### 2.2. 절차

1. 논증을 기호화한다.
    - 전제와 전제 사이는 `,`
    - 전제와 결론 사이는 `/`
    - 예: `J⊃E, ∼J / ∼E`
2. 각 명제 아래에 진리표를 작성한다.
3. 진리표에서
    - **전제가 모두 참이고 결론이 거짓인 줄이 있으면** → **부당**
    - **그런 줄이 하나도 없으면** → **타당**

## 3. 간접 진리표

### 개념

간접 진리표는 모든 경우를 전부 나열하지 않고,  
먼저 **논증이 부당하다고 가정**해서 출발한다.

즉,

- **전제는 모두 참**
- **결론은 거짓**

이라고 놓고, 그 가정이 **모순 없이 유지될 수 있는지**를 본다.

### 절차

1. 논증을 기호화하여 한 줄로 쓴다.
    
2. 이 논증이 부당하다고 가정한다.  
    즉,
    
    - 전제들은 모두 참
    - 결론은 거짓  
        으로 놓고 각 명제의 **주논리연결사** 아래에 진리값을 표시한다.
3. 계산을 진행했을 때
    
    - **모순 없이 끝까지 가능하면** → **부당**
    - **어딘가에서 모순이 생기면** → **타당**

### 자료의 예시

- `∼C⊃(D∨R), ∼B / R⊃C`
- `G⊃(H∨D), H⊃K, G / ∼D⊃K`
- `W⊃P, K⊃P / W⊃Q`
- `∼R⊃A, A⊃R, R⊃∼O / R∙∼O`
- `A⊃(B⊃C), B⊃(A⊃C) / (A∨B)⊃C`

이 예제들은 간접 진리표 방식으로 전제 참 / 결론 거짓을 놓고 추론하는 연습용으로 제시된다.

## 4. 직접 진리표와 간접 진리표의 차이

### 직접 진리표

- 가능한 모든 진리값 경우를 전부 작성
- 절차가 기계적이고 명확함
- 단순명제가 많아지면 표가 매우 커짐

### 간접 진리표

- 전제 참, 결론 거짓이라는 특정 조건에서만 검토
- 경우의 수를 줄일 수 있음
- 모순 유도 방식이므로 더 논리적 해석이 필요함

즉, 직접 진리표는 **전수조사형**, 간접 진리표는 **반례 가능성 검사형**이라고 볼 수 있다.

## 5. 자료의 연습문제 구성

자료 후반부는 직접 진리표와 간접 진리표를 직접 그려 보게 하는 연습문제로 구성된다.

### 5.1 기호화된 논증의 직접 진리표 판별

다음 유형의 문제가 제시된다.

- `~(X∙~Y), ~(Y∙~X) / Y∨X`
- `~(H∙O), O∨~H / ~H`
- `D≡E, F∨~E / F⊃D`
- `B≡(C∨D), ~D∨C / B⊃C`
- `B⊃(O∨R), ~(O∨~B) / B⊃R`

이 문제들은 이미 기호화되어 있으므로, 바로 진리표를 작성해 타당성 여부를 판정하면 된다.

### 5.2 자연어 논증을 번역한 뒤 직접 진리표 판별

자연어 문장을 먼저 기호화한 다음 진리표를 그리는 문제가 나온다.

예시 주제:

- 날씨가 추워지면 상가 수입이 줄어든다는 논증
- 고등학생 학력 저하와 국가 경쟁력, 기업 투명 경영 관련 논증
- 여성 할당제와 역차별 항의, 여성 차별 심화 관련 논증

이 유형은 단순 진리표 계산보다 **정확한 기호화 능력**이 먼저 필요하다.

### 5.3 기호화된 논증의 간접 진리표 판별

간접 진리표 문제로는 다음과 같은 식들이 제시된다.

- `~B∨F, ~B / ~F`
- `A⊃(B∙C), D⊃S / A⊃S`
- `A⊃(X⊃Y), X⊃(Y⊃B) / A⊃(X⊃B)`
- `L⊃(M∨N), N⊃(O∙P), ~M / L⊃~P`
- `A⊃H, H⊃B, ~C⊃A, ~B / C`
- `A⊃(~B⊃~C), C⊃(~B⊃D), (B∨D)⊃E / A⊃E`
- `D∙(E∨F), (D∙F)⊃∼(R∨S), (~R∨~S)⊃~(D∙E) / R≡S`
- `(A∨B)⊃C, C⊃(B∨R), A⊃(~Q⊃B), (Q⊃A)⊃~R / B≡C`
- `(P∨Q)⊃(R∙S), (~P∨~Q)⊃E / (~R∨~S)⊃E`
- `M⊃N, ~O∨P, (N∨P)⊃J, ~J / ~(M∨O)`

후반부로 갈수록 조건문, 논리곱, 논리합, 동치가 결합되어 구조가 복잡해진다.
