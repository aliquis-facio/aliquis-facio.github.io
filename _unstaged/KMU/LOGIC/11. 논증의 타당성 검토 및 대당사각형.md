# 논증의 타당성 검토 및 대당사각형

## 1. 벤 다이어그램을 사용한 논증의 타당성 검토

정언 논리에서 논증의 타당성은 다음 기준으로 판단한다.

> **전제의 벤 다이어그램이 결론의 내용을 이미 포함하고 있으면 타당하다.**
> **전제만으로 결론의 정보가 보장되지 않으면 부당하다.**

예를 들어 다음 논증을 보자.

> 모든 사과는 알칼리성이 강한 과일이다.
> 그러므로 어떤 사과는 알칼리성이 강한 과일이다.

논리식으로 쓰면 다음과 같다.

```text
모든 S는 P이다.
그러므로 어떤 S는 P이다.
```

즉,

```text
A 명제 → I 명제
```

이다.

---

## 2. 현대적 해석과 전통적 해석의 차이

### 2.1 현대적 해석

현대적 해석에서는 **보편명제 A, E가 존재를 함축하지 않는다.**

![697x278](/_image/2026-06-09-14-41-42.jpg)

즉,

```text
모든 S는 P이다.
```

라고 해도 실제로 **S가 존재한다는 뜻은 아니다.**

따라서 다음 추론은 현대적 관점에서는 부당하다.

```text
모든 S는 P이다.
그러므로 어떤 S는 P이다.
```

왜냐하면 전제는 “S가 있다면 P이다”라는 조건만 말할 뿐, 실제로 S가 존재한다는 것을 보장하지 않기 때문이다.

벤 다이어그램으로 보면, A 명제는 `S이면서 P가 아닌 영역`을 비우는 것이지, `S이면서 P인 영역`에 X를 찍는 것이 아니다. 따라서 결론 I 명제의 존재 주장, 즉 `S이면서 P인 것이 적어도 하나 있다`는 내용이 전제에 포함되지 않는다.

---

### 2.2 전통적 해석

전통적 해석에서는 **보편명제가 존재를 함축한다.**

![Pasted image 20260609144614](/_image/2026-06-09-14-46-14.jpg)

즉,

```text
모든 S는 P이다.
```

라고 하면, S가 적어도 하나 존재한다고 본다.

따라서 다음 추론은 전통적 관점에서는 타당하다.

```text
모든 S는 P이다.
그러므로 어떤 S는 P이다.
```

전통적 해석에서는 A 명제가 참이면 S가 존재한다고 보기 때문에, `S이면서 P인 것`도 존재한다고 판단한다. 그래서 A 명제에서 I 명제로 내려가는 추론이 가능하다.

## 3. “거짓이다”의 해석과 모순 관계

| 원래 표현                 | 논리적 의미       |
| --------------------- | ------------ |
| 모든 S는 P라는 것은 거짓이다     | 어떤 S는 P가 아니다 |
| 모든 S는 P가 아니라는 것은 거짓이다 | 어떤 S는 P이다    |
| 어떤 S는 P라는 것은 거짓이다     | 모든 S는 P가 아니다 |
| 어떤 S는 P가 아니라는 것은 거짓이다 | 모든 S는 P이다    |

## 4. 예시: “모든 S는 P가 아니다”에서 “모든 S가 P라는 것은 거짓이다”로

다음 논증을 보자.

> 모든 파렴치한 행위는 도덕적으로 용납되는 일이 아니다.
> 그러므로 모든 파렴치한 행위가 도덕적으로 용납되는 일이라는 것은 거짓이다.

이를 일반화하면 다음과 같다.

```text
모든 S는 P가 아니다.
그러므로 모든 S가 P라는 것은 거짓이다.
```

여기서 전제는 E 명제다.

```text
E: 모든 S는 P가 아니다.
```

결론은 “A가 거짓이다”이므로 O 명제에 해당한다.

```text
A의 부정 = O
어떤 S는 P가 아니다.
```

즉 논증 구조는 다음과 같다.

```text
E → O
```

### 현대적 관점

현대적 관점에서는 E 명제가 존재를 함축하지 않는다.

```text
모든 S는 P가 아니다.
```

라고 해도 S가 실제로 존재한다는 뜻은 아니다.

하지만 결론인 O 명제는 존재를 주장한다.

```text
어떤 S는 P가 아니다.
```

따라서 현대적 관점에서는 전제가 결론의 존재 주장을 포함하지 않으므로 **부당한 논증**이다.

### 전통적 관점

전통적 관점에서는 E 명제가 존재를 함축한다.

```text
모든 S는 P가 아니다.
```

가 참이면 S가 존재하고, 그 S는 P가 아니다.

따라서 O 명제도 따라 나온다.

```text
어떤 S는 P가 아니다.
```

그러므로 전통적 관점에서는 **타당한 논증**이다.

## 5. 대당사각형

대당사각형은 네 가지 정언 명제 사이의 논리적 관계를 보여주는 도식이다.

| 명제   | 형식           | 의미   |
| ---- | ------------ | ---- |
| A 명제 | 모든 S는 P이다    | 보편긍정 |
| E 명제 | 모든 S는 P가 아니다 | 보편부정 |
| I 명제 | 어떤 S는 P이다    | 특칭긍정 |
| O 명제 | 어떤 S는 P가 아니다 | 특칭부정 |

---

## 6. 현대적 관점의 대당사각형

![697x340](/_image/2026-06-09-14-52-31.jpg)

- A와 O, E와 I는 정반대의 진리값을 가진다. 모순관계이다.
- 그 이외에는 어떤 논리적 관계도 성립하지 않는다.

---

## 7. 전통적 관점의 대당사각형

![Pasted image 20260609145458](/_image/2026-06-09-14-54-58.jpg)

- 반대관계: 둘 다 동시에 참일 수 없는 관계 (동시에 거짓일 수는 있음)
- 소반대관계: 둘 다 동시에 거짓일 수 없는 관계 (동시에 참이 될 수는 있음)
- 함축관계: A의 참은 I의 참을 함축, E의 참은 O의 참을 함축

## 9. 연습문제 풀이 방향

### 예시 1

> 도스토예프스키의 모든 소설은 지루한 작품이다.
> 그러므로 도스토예프스키의 어떤 소설은 지루한 작품이다.

논리형식:

```text
A → I
```

판정:

| 관점     | 판정 | 이유             |
| ------ | -- | -------------- |
| 현대적 관점 | 부당 | A는 존재를 함축하지 않음 |
| 전통적 관점 | 타당 | A가 I를 함축함      |

---

### 예시 2

> 모든 재즈 음악가가 아프리카 출신이라는 것은 거짓이다.
> 그러므로 어떤 재즈 음악가는 아프리카 출신이 아니다.

첫 문장은 A 명제의 부정이다.

```text
A가 거짓이다 = O
```

따라서 논증 구조는 다음과 같다.

```text
O → O
```

같은 내용을 반복하는 구조이므로 현대적 관점과 전통적 관점 모두에서 타당하다.

---

### 예시 3

> 어떤 시골 의사가 이타적인 봉사자가 아니라는 것은 거짓이다.
> 그러므로 모든 시골 의사가 이타적인 봉사자가 아니라는 것은 거짓이다.

첫 문장:

```text
어떤 S는 P가 아니다 = O
O가 거짓이다 = A
```

결론:

```text
모든 S는 P가 아니다 = E
E가 거짓이다 = I
```

따라서 논증 구조는 다음과 같다.

```text
A → I
```

판정:

| 관점     | 판정 |
| ------ | -- |
| 현대적 관점 | 부당 |
| 전통적 관점 | 타당 |

---

### 예시 4

> 어떤 결혼식이 엄숙한 행사라는 것은 거짓이다.
> 그러므로 모든 결혼식은 엄숙한 행사라는 것은 거짓이다.

첫 문장:

```text
어떤 S는 P이다 = I
I가 거짓이다 = E
```

결론:

```text
모든 S는 P이다 = A
A가 거짓이다 = O
```

따라서 논증 구조는 다음과 같다.

```text
E → O
```

판정:

| 관점     | 판정 |
| ------ | -- |
| 현대적 관점 | 부당 |
| 전통적 관점 | 타당 |
