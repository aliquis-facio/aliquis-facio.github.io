# OAuth 2.0
## Google API OAuth 2.0 개요
1. Google API Console에서 OAuth 2.0 사용자 인증 정보를 가져옴
2. Google 승인 서버에서 액세스 토큰을 받음
3. 사용자가 부여한 액세스 범위를 검사
4. 액세스 토큰을 API로 전송
5. 필요한 경우 액세스 토큰을 갱신
